For Problems 3–4, write each expression using negative exponents.

3.

  1. \(\displaystyle \dfrac{1}{2^3}\)

  2. \(\displaystyle \dfrac{3}{5^2}\)

  3. \(\displaystyle \dfrac{1}{27}\)

Answer.
4.

  1. \(\displaystyle \dfrac{x}{625}\)

  2. \(\displaystyle \dfrac{2}{z^2}\)

  3. \(\displaystyle \left(\dfrac{z}{10}\right)^5\)
