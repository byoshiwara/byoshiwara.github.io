Caution 5.18.

In Example 5, it is not true that \(8-2\sqrt{7}\) is equal to \(6\sqrt{7}\text{.}\) The order of operations tells us that we must perform the multiplication \(-2\sqrt{7}\) first, then add the result to \(8\text{.}\) You can verify that \(6\sqrt{7} \approx 15.874\text{,}\) which is not the same answer we got in Example 5.
