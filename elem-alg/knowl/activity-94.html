Activity 8.2. Reducing Fractions.

  1. Explain why each calculation is incorrect.

    1. \(\displaystyle \dfrac{x-6}{x-9} \rightarrow \dfrac{x-2}{x-3}\)
    2. \(\displaystyle \dfrac{2x+5}{2} \rightarrow x+5\)
    1. Reduce \(~\dfrac{12x-18}{16x-24}\)
    2. If you evaluate \(~\dfrac{12x-18}{16x-24}~\) for \(x=-17\text{,}\) what do you expect to get?

  3. Reduce \(~\dfrac{x^2-x-6}{x^2-9}\)

  4. Find the opposite of each binomial.

    1. \(\displaystyle 2a+3b\)
    2. \(\displaystyle -x+1\)

  5. Reduce if possible.

    1. \(\displaystyle \dfrac{-x+1}{1-x}\)
    2. \(\displaystyle \dfrac{1+x}{1-x}\)
    3. \(\displaystyle \dfrac{2a-3b}{3b-2a}\)
    4. \(\displaystyle \dfrac{2a-3b}{2b-3a}\)
