The technique used in ExampleÂ 8.29 is called back-substitution. It works in the special case where one of the equations involves exactly one variable, and a second equation involves that same variable and just one other variable. A \(3\times 3\) linear system with these properties is said to be in triangular form. If we can transform a system into triangular form, we can use back-substitution to complete the solution.
