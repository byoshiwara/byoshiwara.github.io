Example 6.20.

Use the substitution \(u = x^3\) to solve the equation
\begin{equation*} x^6 - 4x^3 - 5 = 0 \end{equation*}
Solution.
We set \(u = x^3\text{,}\) so that \(u^2 = \left(x^3\right)^2 = x^6\text{.}\) The original equation then becomes a quadratic equation in the variable \(u\text{,}\) which we can solve by factoring.
\begin{equation*} \begin{aligned}[t] u^2 - 4u - 5 \amp = 0 \amp\amp \blert{\text{Factor the left side.}}\\ (u + 1)(u - 5) \amp = 0 \amp\amp \blert{\text{Apply the zero-factor principle.}}\\ u + 1 \amp = 0~~ \text{ or }~~u - 5 = 0\amp\amp \blert{\text{Solve each equation for }u.}\\ u \amp = -1 ~~ \text{ or }~~ u = 5 \end{aligned} \end{equation*}
Finally, we replace \(u\) by \(x^3\) and solve for \(x\text{.}\)
\begin{equation*} \begin{aligned}[t] x^3 \amp = -1 ~~~~~~~\text{ or } \amp x^3 \amp= 5\amp\amp \blert{\text{Take cube roots.}}\\ x\amp= \sqrt[3]{-1} = -1 ~~~\text{ or } \amp x \amp = \sqrt[3]{5} \end{aligned} \end{equation*}
You can verify that the solutions of the original equation are \(-1\) and \(\sqrt[3]{5}\text{.}\)
