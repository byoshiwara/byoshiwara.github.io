Checkpoint 1.113. Practice 5.

  1. Graph the line \(4x - 2y = 8\) by finding the \(x\)- and \(y\)-intercepts
  2. Compute the slope of the line using the \(x\)-intercept and \(y\)-intercept.
  3. Compute the slope of the line using the points \((4, 4)\) and \((1, -2)\text{.}\)
Solution.
  2. The \(x\)-intercept is \((2,0)\) and the \(y\)-intercept is \((0,-4)\text{.}\)
    The slope is \(m=\dfrac{\Delta y}{\Delta x}= \dfrac{4}{2} = 2\)
  3. The slope is \(m=\dfrac{\Delta y}{\Delta x}= \dfrac{-2-4}{1-4}=\dfrac{-6}{-3} = 2\)
