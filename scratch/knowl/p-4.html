  1. The distance between the base of the ladder and the wall is the side adjacent to the \(75\degree\) angle. We can find the side adjacent to the \(75\degree\) angle angle by using the cosine ratio.
    \begin{align*} \cos (75\degree) \amp = \dfrac {\text{adjacent}}{\text{hypotenuse}}\\ 0.2588 \amp = \dfrac{b}{10}\\ b \amp = 10(0.2588) = 2.588 \end{align*}
    The base of the ladder should be about 2.6 feet from the wall.
  2. To find side how far up the wall the ladder will reach, we could use the Pythagorean theorem now, but it is better to use given information, rather than values we have calculated, to find the other unknown parts. We will use the sine ratio.
    \begin{align*} \sin (75 \degree) \amp = \dfrac {\text{opposite}}{\text{hypotenuse}}\\ 0.9659 \amp = \dfrac{a}{10}\\ b \amp = 10(0.9659) = 9.659 \end{align*}
    The ladder will reach about 9.7 feet.
