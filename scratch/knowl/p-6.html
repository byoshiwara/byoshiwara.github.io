To find side how far up the wall the ladder will reach, we could use the Pythagorean theorem now, but it is better to use given information, rather than values we have calculated, to find the other unknown parts. We will use the sine ratio.
\begin{align*} \sin (75 \degree) \amp = \dfrac {\text{opposite}}{\text{hypotenuse}}\\ 0.9659 \amp = \dfrac{a}{10}\\ b \amp = 10(0.9659) = 9.659 \end{align*}
The ladder will reach about 9.7 feet.
