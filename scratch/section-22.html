Section C.1 Getting Started

Subsection On and Off

If you are reading this textbook in a web browser using a computer with sufficiently wide monitor, you should see button labeled "Calc" just above the text on the page. Clicking on that button will launch a Geogebra calculator applet in the right margin of the book. If the calculator overlaps the text, click on the "Contents" tab above the navigation panel to hide that panel and provide more space for the text. Click again on the "Calc" button to close the calculator app. (And clicking again on the "Contents" tab will re-open the navigation panel.)

Subsection Numbers and Operations

Example C.2.

Compute \(5 - 8\text{.}\)

Enter \(\qquad 5~\)-\(~8\)

and even before we press ⮠, GeoGebra shows the result \(-3\) below the input box.

If we do press ⮠, GeoGebra shows

\(\qquad \begin{aligned}[t] \amp a = 5-8 \\ \amp\rightarrow -3 \end{aligned}\)

in a history region above the input box. There is a scroll bar to the right of the history and the input box.

Example C.3.

Compute \(-5 + 8\text{.}\)

Enter \(\qquad ~5~\)+\(~8\)

to see the result \(3\)

We press ⮠ to tell the calculator to store the computation. We can use the scroll bar to see the input box or the history of previous computations.

The calculator has a key for the value of \(\pi\text{.}\)

Example C.4.

Compute \(2\pi\text{.}\)

Enter \(\qquad 2\) × \(\boxed{\pi\vphantom{1}} \quad\) or \(\quad 2~ \boxed{\pi\vphantom{1}}\)

to get the approximation \(6.2831853071796\)

Subsection Delete and Undo

Press the backspace key (shown in the keyboard image C.1) to delete the character before the cursor or any highlighted characters.

To remove an item from the history, tap the kebab menu icon (three vertically stacked dots) to the right of the item, then tap "Delete".

To remove the most recent item in the history, tap the Undo icon in the Graphics View. (The icon looks like an arched arrow pointing left.)
