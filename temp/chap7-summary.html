  • percent growth rate

  • exponential decay

  • decay factor

  • exponential function

Subsection Key Concepts

  1. Exponential growth was modeled by increasing functions of the form \(P(t)=P_0b^t\text{,}\) where the growth factor, \(b\text{,}\) is a number greater than 1.
  2. Exponential Growth.

    The function

    \begin{equation*} P(t)=P_0 b^t \end{equation*}

    describes exponential growth, where \(P_0=P(0)\) is the initial value of the function and \(b\) is the growth factor.

  3. Growth by a Constant Percent.

    The function

    \begin{equation*} P(t) =P_0 (1+r)^t \end{equation*}

    describes exponential growth at a constant percent rate of growth, \(r\text{.}\)

    The initial value of the function is \(P_0 = P(0)\text{,}\) and \(b=1+r\) is the growth factor.

  4. Compound Interest.

    If a principal of \(P\) dollars is invested in an account that pays an interest rate \(r\) compounded annually, the balance \(B\) after \(t\) years is given by

    \begin{equation*} B=P(1+r)^t \end{equation*}
  5. Laws of Exponents.

    1. \(\displaystyle \displaystyle{a^m\cdot a^n = a^{m+n}}\)

    2. \(\displaystyle \displaystyle{\frac{a^m}{a^n}=a^{m-n}}\)

    3. \(\displaystyle \displaystyle{\left(a^m\right)^n=a^{mn}}\)

    4. \(\displaystyle \displaystyle{\left(ab\right)^n=a^n b^n}\)

    5. \(\displaystyle \displaystyle{\left(\frac{a}{b}\right)^n=\frac{a^n}{b^n}}\)

  6. If \(0 \lt b \lt 1\text{,}\) then \(P(t)=P_0b^t\) is a decreasing function. In this case \(P(t)\) is said to describe exponential decay.
  7. A percent increase of \(r\) (in decimal form) corresponds to a growth factor of \(b=1+r\text{.}\) A percent decrease of corresponds to a decay factor of \(b=1-r\text{.}\)
  8. Exponential Growth and Decay.

    The function

    \begin{equation*} P(t) = P_0 b^t \end{equation*}

    models exponential growth and decay.

    • \(P_0 =P(0)\) is the initial value of \(P\text{;}\)
    • \(b\) is the growth or decay factor.

    1. If \(b \gt 1\text{,}\) then \(P(t)\) is increasing, and \(b = 1 + r\text{,}\) where \(r\) represents percent increase.

    2. If \(0 \lt b \lt 1\text{,}\) then \(P(t)\) is decreasing, and \(b = 1 - r\text{,}\) where \(r\) represents percent decrease.

  9. Exponential Function.

    An exponential function has the form

    \begin{equation*} f(x) = ab^x,~~~~ \text{ where } ~~~b \gt 0 ~~~\text{ and } ~~~b \ne 1 \text{, } ~~~a \ne 0 \end{equation*}
  10. Exponential Notation for Radicals.

    For any integer \(n \ge 2\) and for \(a \ge 0\text{,}\)

    \begin{equation*} a^{1/n} = \sqrt[n]{a} \end{equation*}
  11. The growth factor of an exponential function is analogous to the slope of a linear function: Each measures how quickly the function is increasing (or decreasing).
  12. Suppose \(L(t)=a+mt\) is a linear function and \(E(t)=ab^t\) is an exponential function. For each unit that \(t\) increases, \(m\) units are added to the value of \(L(t)\text{,}\) whereas the value of \(E(t)\) is multiplied by \(b\text{.}\)
  13. We do not allow the base of an exponential function to be negative, because if \(b \lt 0\text{,}\) then \(b^x\) is not a real number for some values of \(x\text{.}\)
  14. Properties of Exponential Functions, \(f(x) = ab^x\text{,}\) \(a \gt 0\).

    1. If \(b \gt 1\text{,}\) the function is increasing and concave up;

      if \(0 \lt b \lt 1\text{,}\) the function is decreasing and concave up.

    2. The \(y\)-intercept is \((0,a)\text{.}\) There is no \(x\)-intercept.

  15. The negative \(x\)-axis is a horizontal asymptote for exponential functions with \(b \gt 1\text{.}\) For exponential functions with \(0 \lt b \lt 1\text{,}\) the positive \(x\)-axis is an asymptote.
  16. Exponential functions are not the same as the power functions we studied earlier. Although both involve expressions with exponents, it is the location of the variable that makes the difference.
  17. Power Functions vs Exponential Functions.

    \(\hphantom{General formula and m}\)

    Power Functions

    Exponential Functions

    General formula

    \(h(x)=kx^p\)

    \(f(x)=ab^x\)

    Description

    variable base and constant exponent

    constant base and variable exponent

    Example

    \(h(x)=2x^3\)

    \(f(x)=2(3^x)\)
  18. Many exponential equations can be solved by writing both sides of the equation as powers with the same base. If two equivalent powers have the same base, then their exponents must be equal also (as long as the base is not 0 or \(\pm 1\)).
  19. Definition of Logarithm.

    For \(b\gt 0, b\ne 1\text{,}\) the base \(b\) logarithm of \(x\), written \(\log_{b} x\text{,}\) is the exponent to which \(b\) must be raised in order to yield \(x\text{.}\)

  20. Logarithms and Exponents: Conversion Equations.

    If \(b \gt 0\text{,}\) \(b\ne 1\text{,}\) and \(x \gt 0\text{,}\)

    \begin{equation*} \blert{y = \log_b x}~~~ \text{ if and only if }~~~ \blert{ x = b^y} \end{equation*}
  21. Some Useful Logarithms.

    For any base \(b \gt 0, b\ne 1\text{,}\)

    \begin{align*} \log_b b \amp = 1~~~ \text{ because } ~~~b^1 = b\\ \log_b 1 \amp = 0 ~~~ \text{ because } ~~~b^0 = 1\\ \log_b{b^x} \amp = x~~~ \text{ because } ~~~b^x = b^x \end{align*}
  22. We use logarithms to solve exponential equations, just as we use square roots to solve quadratic equations. The operation of taking a base \(b\) logarithm is the inverse of raising the base \(b\) to a power, just as extracting square roots is the inverse of squaring a number.
  23. Steps for Solving Base 10 Exponential Equations.
    1. Isolate the power on one side of the equation.
    2. Rewrite the equation in logarithmic form.
    3. Use a calculator, if necessary, to evaluate the logarithm.
    4. Solve for the variable.
  24. Properties of Logarithms.

    If \(x,~y,\) and \(b\gt 0,\) and \(b\ne 1\text{,}\) then

    1. \(\displaystyle \log_b {xy} = \log_b {x} + \log_b {y}\)
    2. \(\displaystyle \log_b {\dfrac{x}{y}} = \log_b {x} - \log_b {y}\)
    3. \(\displaystyle \log_b {x^k} = k\log_b {x} \)
  25. Steps for Solving Exponential Equations.
    1. Isolate the power on one side of the equation.
    2. Take the base 10 logarithm of both sides of the equation.
    3. Apply the third property of logarithms to simplify.
    4. Use a calculator, if necessary, to evaluate the logarithm.
    5. Solve for the variable.

    Section 7.4 Chapter 7 Summary and Review

    Subsection Key Concepts

    1. Changes to the amplitude, period, and midline of the basic sine and cosine graphs are called transformations. Changing the midline shifts the graph vertically, changing the amplitude stretches or compresses the graph vertically, and changing the period stretches or compresses the graph horizontally.
    2. The order in which we apply transformations to a function makes a difference in the graph.
    3. Amplitude, Period, and Midline of Sinusoidal Functions.
      1. The graph of
        \begin{equation*} y=A\cos x ~~~~\text{or}~~~~ y=A\sin x \end{equation*}
        has amplitude \(\abs A\text{.}\)
      2. The graph of
        \begin{equation*} y=\cos Bx ~~~~\text{or}~~~~ y=\sin Bx \end{equation*}
        has period \(\dfrac{2\pi}{B}\text{.}\)
      3. The graph of
        \begin{equation*} y=k+\cos x ~~~~\text{or}~~~~ y=k+\sin x \end{equation*}
        has midline \(y=k\text{.}\)
    4. Horizontal Shifts.

      The graphs of

      \begin{equation*} y = \sin (x-h)~~~~ \text{and}~~~~ t = \cos (x-h) \end{equation*}

      are shifted horizontally compared to the graphs of \(y = \sin x\) and \(y = \cos x\text{.}\)

      • If \(h \gt 0\text{,}\) the graph is shifted to the right.
      • If \(h \lt 0\text{,}\) the graph is shifted to the left.
    5. Standard Form for Sinusoidal Functions.

      The graphs of the functions

      \begin{equation*} y = A\sin B(x-h) + k~~~~ \text {and}~~~~ y = A\cos B(x-h) + k \end{equation*}

      are transformations of the sine and cosine graphs.

      1. The amplitude is \(\abs{A}\text{.}\)
      2. The midline is \(y = k\text{.}\)
      3. The period is \(\dfrac{2\pi}{\abs{B}},~~B \not= 0\text{.}\)
      4. The horizontal shift is \(h\) units to the right if \(h\) is positive, and \(h\) units to the left if \(h\) is negative.
    6. Solutions of Trigonometric Equations.
      1. The equation \(\cos \theta = k, ~~ -1\lt k \lt 1\text{,}\) has two solutions between \(0\) and \(2\pi\text{:}\)
        \begin{equation*} \theta_{1}=\cos^{-1}(k)~~ \text{and}~~ \theta_{2}=2\pi -\theta_{1} \end{equation*}
      2. The equation \(\sin \theta = k, ~~ -1\lt k \lt 1\text{,}\) has two solutions between \(0\) and \(2\pi\text{:}\)
        \begin{equation*} \text{If}~k \gt 0:~~ \theta_{1} =\sin^{-1}(k)~~ \text{and}~~ \theta_{2}=\pi -\theta_{1} \end{equation*}
        \begin{equation*} \text{If}~k \lt 0:~~ \theta_{1}=\sin^{-1}(k) + 2\pi ~~ \text{and}~~ \theta_{2}=\pi -\sin^{-1}(k) \end{equation*}
      3. The equation \(\tan \theta = k\text{,}\) has two solutions between \(0\) and \(2\pi\text{:}\)
        \begin{equation*} \text{If}~k \gt 0:~~ \theta_{1}=\tan^{-1}(k)~~ \text{and}~~ \theta_{2}=\pi +\theta_{1} \end{equation*}
        \begin{equation*} \text{If}~k \lt 0:~~ \theta_{1}=\tan^{-1}(k) + \pi ~~ \text{and}~~ \theta_{2}=\pi +\theta_{1} \end{equation*}
    7. If \(n\) is a positive integer, the equations \(\sin n\theta = k\) and \(\cos n\theta = k\) each have \(2n\) solutions between \(0\) and \(2\pi\text{,}\) for \(-1\lt k \lt 1\text{.}\)
    8. The equation \(\tan n\theta = k\) has one solution in each cycle of the graph.
    9. Using a Substitution to Solve Trigonometric Equations.

      To solve the equation \(\sin (Bx+C)=k\) or \(\cos (Bx+C)=k\text{:}\)

      1. Substitute \(\theta = Bx+C\text{,}\) and find two solutions for \(\sin \theta = k\) or \(\cos \theta = k\text{.}\)
      2. Replace \(\theta\) by \(Bx+C\) in each solution, and solve for \(x\text{.}\)
      3. Find the other solutions by adding multiples of \(\dfrac{2\pi}{B}\) to the first two solutions.

      To solve the equation \(\tan (Bx+C)=k\text{:}\)

      1. Substitute \(\theta = Bx+C\text{,}\) and find one solution for \(\tan \theta = k\text{.}\)
      2. Replace \(\theta\) by \(Bx+C\) and solve for \(x\text{.}\)
      3. Find the other solutions by adding multiples of \(\dfrac{\pi}{B}\) to the first solution.
