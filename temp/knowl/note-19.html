<<<<<<< HEAD Note 7.21.
  • We do not allow \(b\) to be negative, because if \(b \lt 0\text{,}\) then \(b^x\) is not a real number for some values of \(x\text{.}\) For example, if \(b = -4\) and \(f (x) = (-4)^x\text{,}\) then \(f (1/2) = (-4)^{1/2}\) is an imaginary number.
  • We also exclude \(b = 1\) as a base because \(1^x = 1\) for all values of \(x\text{;}\) hence the function \(f (x) = 1^x\) is actually the constant function \(f (x) = 1\text{.}\)
in-context ======= Note 4.39.

Recall that the slope of a line is given by the ratio \(\dfrac {\text{change in}~ y}{\text{change in}~x}\) as we move from one point to another on the line. So, if we create a right triangle by dropping a perpendicular segment from the line to the \(x\)-axis, the ratio of sides \(\dfrac {\text{opposite}}{\text{adjacent}}\) gives the slope of the line.
