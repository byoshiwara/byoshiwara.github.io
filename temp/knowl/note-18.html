In the preceding Example, the value of the computer equipment decreases by 37% each year, so 63% of the value remains, and the decay factor for the value function is 0.63, not 0.37. The function \(V(t)\) gives the value remaining, not the amount that has depreciated.

in-context ======= Note 4.20.

In the previous example we found two solutions of the equation \(\sin \theta = -0.6428\text{.}\) Actually, there are infinitely many solutions -- namely, all the angles coterminal with \(320\degree\) or \(220\degree\text{.}\) Some of these angles are listed below.

\begin{equation*} \begin{aligned}[t] \text{Angles coterminal with}~ 220\degree: \amp~~ \cdots, ~-500\degree, ~-140\degree, ~580\degree, ~940\degree, ~\cdots\\ \text{Angles coterminal with}~ 320\degree: \amp~~ \cdots, ~-400\degree, ~-40\degree, ~680\degree, ~1040\degree, ~\cdots\\ \end{aligned} \end{equation*}

Because we can easily find coterminal solutions by adding or subtracting multiples of \(360\degree\text{,}\) when solving an equation we usually list only the solutions between \(0\degree\) and \(360\degree\text{.}\)
