in-context =======

This calculation is similar to the product \((4x - 1)(3x+2)\text{,}\) except that the variable \(x\) has been replaced by \(\sin C\text{.}\) Compare the calculations for the two products; first the familiar algebraic product:

\begin{equation*} \begin{aligned}[t] \blert{(4x - 1)(3x + 2)} \amp \blert{= 4x\cdot 3x + 4x\cdot 2 - 1\cdot 3x - 1\cdot 2}\\ \amp \blert{= 12x^2 + 8x - 3x - 2 = 12x^2 + 5x - 2}\\ \end{aligned} \end{equation*}

We compute the product in this example in the same way, but replacing \(x\) by \(\sin C\text{.}\)

\begin{equation*} \begin{aligned}[t] (4\sin C - 1)(\sin C + 2) \amp = (4\sin C)(3\sin C) + (4\sin C)\cdot 2 - 1(3\sin C) - 1\cdot 2\\ \amp = 12\sin^2 C + 5\sin C - 2\\ \end{aligned} \end{equation*}
