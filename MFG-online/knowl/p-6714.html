If

\begin{equation*} z = kxy \text{, } ~~ k \gt 0 \end{equation*}

we say that \(z\) varies jointly with \(x\) and \(y\text{.}\) If

\begin{equation*} z = k\frac{x}{y}\text{, } ~~ k \gt 0\text{, } ~~ y \ne 0 \end{equation*}

we say that \(z\) varies directly with \(x\) and inversely with \(y\text{.}\)
