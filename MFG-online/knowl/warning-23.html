Caution6.95
In Example 6.93, the solution of the inequality \(x^2 - 4 \gt 0\) is the set \begin{equation*} x \lt -2 ~~~\text{ or }~~~x \gt 2. \end{equation*} This set is another type of compound inequality, and its graph consists of two pieces, as shown in Figure 6.96. The left piece of the set is \(x \lt -2\text{,}\) and the right piece is \(x \gt 2\text{.}\) It would be incorrect to describe the solution set as \(-2 \gt x \gt 2\text{,}\) because this notation implies that \(-2 \gt 2\text{.}\) We must write the solution as two parts: \(x \lt -2\) or \(x \gt 2\text{.}\)
