Exercise58

A farmer inherits an apple orchard on which \(60\) trees are planted per acre. Each tree yields \(12\) bushels of apples. Experimentation has shown that for each tree removed per acre, the yield per tree increases by \(12\) bushel.

  1. Write algebraic expressions for the number of trees per acre and for the yield per tree if \(x\) trees per acre are removed.

  2. Write a formula for the total yield per acre as a function of \(x\text{.}\)

  3. What is the maximum yield per acre that can be achieved by removing trees? How many trees per acre should be removed to achieve this yield?

  4. How many trees should be removed per acre in order to harvest at least \(850\) bushels per acre?

  5. Graph the yield function and use the graph to verify your answers to parts (c) and (d).
