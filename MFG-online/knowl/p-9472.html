To use the graph of \(h\) to find values of \(h^{-1}\text{,}\) we start with the vertical axis and find the point on the graph with \(C = 10\text{.}\) This point is \((50, 10)\text{,}\) so \(F = 50\) when \(C = 10\text{,}\) or \(h^{-1}(10) = 50\text{.}\) When the Celsius temperature is \(10\)°, the Fahrenheit temperature is \(50\)°.
