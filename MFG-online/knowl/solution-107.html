Solution4.49.1

  1. To evaluate \(\log_3 9\text{,}\) we ask what exponent on base \(3\) will produce \(9\text{.}\) In symbols, we want to fill in the blank in the equation \(3^{\underline{?}} = 9\text{.}\) The exponent we need is \(2\text{,}\) so

    \begin{equation*} \log_3 9 = 2 \text{ because } 3^2 = 9 \end{equation*}

    We use similar reasoning to compute the other logarithms.

  2. \(\log_5{125} = 3 \text{ because } 5^3 = 125\)

  3. \(\log_4{\dfrac{1}{16}}= -2 \text{ because } 4^{-2} = \dfrac{1}{16}\)

  4. \(\log_5{\sqrt{5}} = \dfrac{1}{2} \text{ because } 5^{1/2} =\sqrt{5}\)
