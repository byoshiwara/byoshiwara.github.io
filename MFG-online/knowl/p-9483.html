We can use a graph of a function \(y = f (x)\) to find values of the inverse function \(x = f^{-1}(y)\text{.}\) FigureÂ 5.13 shows the graph of \(f(x) = x^3 + 2\text{.}\) You already know how to evaluate a function from its graph: We start with the horizontal axis. For instance, to evaluate \(f(-2)\text{,}\) we find \(-2\) on the \(x\)-axis, move vertically to the point on the graph with \(x = -2\text{,}\) in this case \((-2, -6)\text{,}\) and read the \(y\)-coordinate of the point. We see that \(f(-2) = -6\text{.}\)
