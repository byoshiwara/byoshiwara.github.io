So far, we have only considered examples in which \(\Delta x\) and \(\Delta y\) are positive numbers, but they can also be negative.

\begin{equation*} \Delta x = \begin{cases} \text{positive if } x \text{ increases (move to the right)}\\ \text{negative if } x \text{ decreases (move to the left)}\\ \end{cases} \end{equation*} \begin{equation*} \Delta y = \begin{cases} \text{positive if } y \text{ increases (move up)}\\ \text{negative if } y \text{ decreases (move down)}~~~~~~~~~\\ \end{cases} \end{equation*} in-context
