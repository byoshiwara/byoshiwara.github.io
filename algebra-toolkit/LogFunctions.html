Subsection 1. Estimate logs

Subsubsection Examples

Example 5.14.

Write each log equation in exponential form. Then use trial and error to estimate the log between two integers.

  1. \(\displaystyle \log_2 6=x\)
  2. \(\displaystyle \log_2 24=x\)
  3. \(\displaystyle \log_2 100=x\)
  4. \(\displaystyle \log_2 0.3=x\)
Solution
  1. \(2^x=6.~\) Because \(~2^2=4~\) and \(~2^3=8\text{,}\) \(~2 \lt x \lt 3\text{.}\)
  2. \(2^x=24.~\) Because \(~2^4=16~\) and \(~2^5=32\text{,}\) \(~4 \lt x \lt 5\text{.}\)
  3. \(2^x=100.~\) Because \(~2^6=64~\) and \(~2^7=128\text{,}\) \(~6 \lt x \lt 7\text{.}\)
  4. \(2^x=0.3.~\) Because \(~2^{-2}=\dfrac{1}{4} = 0.25~\) and \(~2^{-1}=\dfrac{1}{2} = 0.5\text{,}\) \(~-2 \lt x \lt -1\text{.}\)

Subsection 2. Use function notation

Subsubsection Examples

Example 5.18.

\(f(x)=\log_6 x\)

  1. What is \(~f^{-1}(x)\text{?}\)
  2. Evaluate and simplify \(~f(f^{-1}(4))\)
  3. Evaluate and simplify \(~f^{-1}(f(5))\)
Solution
  1. \(\displaystyle ~f^{-1}(x) = 6^x\)
  2. \(\displaystyle ~f(f^{-1}(4)) = f(6^4) = \log_6 (6^4) = 4\)
  3. \(\displaystyle ~f^{-1}(\log_6 5) = 6^{\log_6 5} = 5\)
Example 5.19.

For each function \(~f(x)~\text{,}\) decide whether \(f(a+b)=f(a)+f(b)\text{.}\)

  1. \(\displaystyle f(x)=3^x\)
  2. \(\displaystyle f(x)=\log_3 x\)
Solution

  1. \(f(a+b)=3^{a+b},~\) and \(~f(a)+f(b) = 3^a+3^b.~\)

    But \(~3^{a+b}~\) is not equivalent to \(~3^a+3^b;~\) in fact \(~3^{a+b} =3^a \cdot 3^b.\)

  2. \(f(a+b)=\log_3 (a+b),~\) and \(~f(a)+f(b) = \log_3 a +\log_3 b.~\)

    But \(~\log_3 (a+b)~\) is not equivalent to \(~\log_3 a +\log_3 b;~\) in fact \(~\log_3 (ab) = \log_3 a +\log_3 b.\)
