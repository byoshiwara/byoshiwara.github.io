Section 3.5 Joint Variation

Subsection Functions of Two or More Variables

So far, we have studied functions that relate values of an output variable to values of a single input variable. But it is not uncommon for an output variable to depend on two or more inputs. Many familiar formulas describe functions of several variables.

For example, the perimeter of a rectangle depends on its length and width. The volume of a cylinder depends on its radius and height. The distance you travel depends on your speed and the time you spent traveling. Each of these formulas can be written with function notation.

\begin{equation*} \begin{aligned}[t] \amp P = f(l,w) = 2l + 2w \amp\amp \text{Perimeter is a function of length and width.}\\ \amp V = f(r, h) = \pi r^2h \amp\amp \text{Volume is a function of radius and height.}\\ \amp d = f(r, t) = rt \amp\amp \text{Distance is a function of rate and time.} \end{aligned} \end{equation*}
Example 3.61.

The cost, \(C\text{,}\) of driving a rental car is given by the function

\begin{equation*} C = f(t, g) = 29.95t + 2.80g \end{equation*}

where \(t\) is the number of days you rent the car and \(g\) is the number of gallons of gas you buy.

  1. Evaluate \(f(3,10)\) and explain what it means.

  2. You have $100 to rent a car for \(2\) days. How much gas can you buy?

Solution

  1. We substitute \(3\) for \(t\) and \(10\) for \(g\) to find

    \begin{equation*} f(3, 10) = 29.95(3) + 2.80(10) = 117.85 \end{equation*}

    It will cost $\(117.85\) to rent a car for \(3\) days and buy \(10\) gallons of gas.

  2. We would like to find the value of \(g\) when \(t = 2\) and \(C = 100\text{.}\) That is, we want to solve the equation

    \begin{equation*} \begin{aligned}[t] 100 \amp = f(2, g) = 29.95(2) + 2.80g \\ 100 \amp = 59.90 + 2.80g \\ g \amp = 14.32 \end{aligned} \end{equation*}

    You can buy \(14.32\) gallons of gas.

Subsection Tables of Values

Just as for functions of a single variable, we can use tables to describe functions of two variables, \(z = f (x, y)\text{.}\) The row and column headings show the values of the two input variables, and the table entries show the values of the output variable.

Example 3.64.

Windchill is a function of two variables, temperature and wind speed, or \(W = f(s, t)\text{.}\) The table shows the windchill factor for various combinations of temperature and wind speed.

\(5\) \(33\) \(27\) \(21\) \(16\) \(12\) \(7\) \(0\) \(-5\)
\(10\) \(22\) \(16\) \(10\) \(3\) \(-3\) \(-9\) \(-15\) \(-22\)

  1. What is the windchill factor when the temperature is \(15\degree\)F and the wind is blowing at \(20\) mph? Write this fact with function notation.

  2. Find a value for \(t\) so that \(f (10, t) =-15\text{.}\) What does this equation tell you about the windchill factor?

  3. Solve the equation \(f (s, 30) = 1\text{.}\) What does this tell you about the windchill factor?

Solution

  1. We look in the row for \(20\) mph and the column for \(15\degree\text{.}\) The associated windchill factor is \(-17\text{,}\) so \(f(20, 15) = -17\text{.}\)

  2. We look in the row for \(s = 10\) until we find the windchill factor of \(W = -15\text{.}\) The column heading for that entry is \(5\text{,}\) so \(t = 5\text{.}\) When the wind speed is \(10\) mph and the windchill factor is \(-15\text{,}\) the temperature is \(5\degree\)F.

  3. In the \(t = 30\degree\)F column, we find the windchill factor of \(1\) in the \(25\)-mph row, so \(s = 25\text{.}\) The wind speed is \(25\) mph when the temperature is \(30\degree\)F and the windchill factor is \(1\text{.}\)

Subsection Joint Variation

Sometimes we can find patterns relating the entries in a table.

Example 3.66.

Rectangular beams of a given length can support a load, \(L\text{,}\) that depends on both the width and the depth of the beam, so that \(L = f (w, d)\text{.}\) The table shows some of the values.

  1. Evaluate the function at \(f(2, 5)\text{.}\) Interpret your answer for the problem situation.

  2. Is it true that \(f(2, 5) = f(5, 2)\text{?}\)

  3. Consider the row corresponding to a width of \(3\) cm. How does the load depend on the depth?

Solution

  1. In the row for \(2\) cm and the column for \(5\) cm, we find that \(f (2, 5) = 500\text{.}\) A beam of width \(2\) cm and depth \(5\) cm can support a maximum load of \(500\) kilograms.

  2. In the row for \(5\) cm and the column for \(2\) cm, we find that \(f (5, 2) = 200\text{,}\) so \(f (2, 5) \ne f (5, 2)\text{.}\)

  3. Using the row for width \(3\) cm, we make a new table showing the relationship between load and depth. The increase in load for each increase of \(1\) cm in depth is not a constant, so the graph shown below is not a straight line.

    The curve does pass through the origin, so perhaps the data describe direct variation with a power of depth. If we try the equation \(L = kd^2\) and use the point \((1, 30)\text{,}\) we find that \(30 = k \cdot 1^2\text{,}\) so \(k = 30\text{.}\) You can check that the equation \(L = 30d^2\) does fit the rest of the data points.

In Notebook 3.67, you should find that the load varies directly with width when the depth is \(3\) centimeters. In fact, the load varies directly with width for any fixed depth.

In Example 3.66, we saw that the load varies with the square of depth when the width is \(3\) centimeters, and this relationship also holds for any value of \(w\text{.}\) Consequently, we can find a constant \(k\) such that

\begin{equation*} \text{load} = k \cdot \text{width} \cdot \text{depth}^2 \end{equation*}

This relationship between variables is an example of joint variation.

Notebook 3.68. QuickCheck 2.
Example 3.69.

Find a formula for load as a function of width and depth for the data in Example 3.66.

Solution

The function we want has the form

\begin{equation*} L = f (w, d) = kwd^2 \end{equation*}

for some value of \(k\text{.}\) We use the fact that \(L = 10\) when \(w = 1\) and \(d = 1\text{.}\) Then

\begin{equation*} 10 = k(1)(1^2) \end{equation*}

so \(k = 10\text{.}\) The formula for load as a function of width and depth is

\begin{equation*} L = 10wd^2 \end{equation*}

You can check that this formula works for all the values in the table.

Subsection Graphs

It is possible to make graphs in three dimensions for functions of two variables, but we will not do that here. Instead, we will represent such functions graphically by holding one of the two variables constant.

Example 3.72.

In Example 3.69, we found a formula for the load a beam can support,

\begin{equation*} L = 10wd^2 \end{equation*}

  1. Graph \(L\) as a function of \(w\) for \(d = 1, 2, 3,\) and \(4\text{.}\)

  2. Graph \(L\) as a function of \(d\) for \(w = 1, 2, 3,\) and \(4\text{.}\)

Solution

  1. We make four graphs on the same grid, one for each value of \(d\text{:}\)

    \begin{equation*} \begin{aligned}[t] \text{when } d \amp= 1, \amp\amp L = 10w \\ \text{when } d \amp= 2, \amp\amp L = 40w \\ \text{when } d \amp= 3, \amp\amp L = 90w \\ \text{when } d \amp= 4, \amp\amp L = 160w \end{aligned} \end{equation*}

    The graphs are shown at left below. We can see that \(L\) varies directly with the width of the beam for any fixed value of its depth.

  2. We make one graph for each value of \(w\text{:}\)

    \begin{equation*} \begin{aligned}[t] \text{when } w \amp= 1, \amp\amp L = 10d^2 \\ \text{when } w \amp= 2, \amp\amp L = 20d^2 \\ \text{when } w \amp= 3, \amp\amp L = 30d^2 \\ \text{when } w \amp= 4, \amp\amp L = 40d^2 \end{aligned} \end{equation*}

    The graphs are shown at right above. For any fixed value of its width, \(L\) varies directly with the square of depth.
