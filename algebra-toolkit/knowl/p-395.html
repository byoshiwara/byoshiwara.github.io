To solve the equation, we want to find \(x\)-values that produce a function value of 8. Each point on the graph has coordinates \((x,f(x))\) for a different value of \(x\text{,}\) and the vertical coordinate of each point is given by the function value, \(f(x)\text{.}\) So we look for points on the graph with vertical coordinate \(f(x)=8\text{.}\)
