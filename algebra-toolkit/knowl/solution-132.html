Solution 4.57.1.

This is an exponential equation. We divide both sides by 1.5, then take logarithms.

\begin{equation*} \begin{aligned}[t] \log 3^{x/5} \amp = \log 8 \amp\amp \blert{\text{Apply the third log property.}}\\ \dfrac{x}{5} \log 3 \amp = \log 8\\ x \amp \dfrac{5\log 8}{\log 3} = 9.46 \end{aligned} \end{equation*}
