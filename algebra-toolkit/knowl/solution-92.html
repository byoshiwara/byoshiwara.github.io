Solution 4.29.1.
  1. \(3^4 = 81~\text{,}\) so \(~x=4\text{.}\)
  2. \(5^3 = 125~\text{,}\) so \(~5^{-3} = \dfrac{1}{125}~\text{,}\) and \(~x=-3\text{.}\)
  3. \((\dfrac{3}{4})^2 = \dfrac{9}{16}~\text{,}\) so \(~(\dfrac{3}{4})^{-2} = \dfrac{16}{9}~~\text{,}\) and \(~x=-2\text{.}\)
  4. \(64^{1/3} = 4~\text{,}\) and \(~4^2=16~\text{,}\) so \(~64^{2/3} = (64^{1/3})^2 = 4^2 = 16~\text{,}\) and \(~x=\dfrac{2}{3}\text{.}\)
