If you don't have Geometer's Sketchpad, you might want to try one of the following. Both of these (FREE) dynamic geometry programs include not only the simple construction tools needed to duplicate the conic section applets, but also the authoring tools to create applets embedded in web pages.

GeoGebra (which also has a dynamic algebra window showing equations of the geometric objects)
Geometry Explorer (which also includes dynamic non-Euclidean geometry tools)
