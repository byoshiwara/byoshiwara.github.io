Interactive Lessons in Excel

Examples

My favorite interactive lesson was created by my colleague Bob Martinez. He wrote "Terminate the Terminator" in Mathcad, and I translated it to the Excel file terminator.xls.

You can certainly create interactive lessons without any special bells and whistles, but you sometimes have to work to make life easy for the user.

Several of the interactive lessons I've created use macros to determine the action of clicking on a button, etc. Excel automatically disables macros under High (the default) security level. To be given the option of accepting or rejecting macros, click on Tools->Macro->Security..., and chooose Medium.

The polar.xls file can be considered a weakly interactive lesson, as it demonstrates the use of "option buttons" (a.k.a "radio buttons"). I first played with these controls (or widgets) in buttons.xls. I also briefly looked at a more sophisticated set of controls in compare.xls.

You can see the other uses of controls in the files riemann.xls, grapher.xls, rowops.xls, and fractions.xls (the last file requires loading the Analysis TookPak from the Excel installation disk).

Creating your own basic controls is fairly simple. First make sure that the necessary control icons are available in Excel. Press View->Toolbars and make sure that the "Forms" box is checked. (We are NOT using the "Control Toolbox" icons, which are more versatile but slightly more effort to set up.)

The controls icons should look something like this.

When you position the cursor above one of the control icons (while that window is active), the name of the control will be displayed.

 

Scroll Bars and Spinners

Scroll (or slider) bars and spinners give the user an easy way to vary a parameter. These two types of controls are used side-by-side in the file buttons.xls. The scroll bar can be oriented either vertically or horizontally, but the spinner is oriented only up and down. (Excel may eventually allow other orientations).

When oriented vertically, the scroll bar is increasing downwards, while the spinner is increasing upwards. In the riemann.xls file, a scroll bar is provided so that the user can use the mouse to vary the number of intervals by clicking or dragging.

We will create both a slider bar and a spinner to compare and contrast the two similar control types.

Creating scroll bars

Open a new file. Click on the Scroll Bar icon. (It consists of a stack of two squares, with an up arrow in the upper square, and a down arrow in the lower square.) Drag a short, wide rectangle to get a horizontal scroll bar. If you don't like the shape, you can drag a handle to re-shape the scroll bar.

When the rectangle is wider than tall, the scroll bar is horizontal; if the rectangle is taller than wide, the scroll bar is vertical. You can also drag the scroll bar around. If you lose the handles, right-click on the slider, then press ESC.

Right-click on the scroll bar, then select "Format Control".Select the "Control" tab. Change the "Current value" to 50, check that the Minimum value is 0, the Maximum value is 100, the Incremental change is 1, and the Page change is 10.

Click on the button at the right of the "Cell link:" box. Click on the box D1, then close the pop-up Format Control box. Next, click on OK to close the Format Object window.

Select cell D3 and type

=(d1-50)/10

and Enter.

Try clicking on the scroll bar or dragging the tab in its middle.

Creating a Spinner

Click on the Spinner icon. (It should be next to the scroll bar icon, and it consists of two up and down triangles separted by a horizontal line.) Drag a tall, thin rectangle to get a spinner. Re-shape as desired.

Right-click on the spinner, then select "Format Control". Set the "Current value" to 10, the Minimum value to 0, the Maximum value to 20, the Incremental change to 1, and the Cell link to $F$1. Click on OK to close the Format Object window.

Select cell F3 and type

=(f1-10)/2

and Enter.

Enter -10 in A1 and -9.5 in A2. Select both cells, then drag the lower right corner of A2 down to A41. In cell B1 enter

=d$3*a1+f$3

then select B1 and double-click on the lower right corner of the cell to "black-cross copy" the contents of that cell into B2:B41.

Select the data in A1:B41 and create an XY (Scatter) graph: Click on the chart icon, select the XY (Scatter) icon and the "Scatter with data points connected by smoothed Lines without markers" sub-type.

After clicking Next, select the Series tab and check that the X Values are from column A and the Y Values are from column B. (The name of the current sheet, probably "Sheet1," is also shown in the input boxes.) Click Next.

Make the Chart title

y = ax + b

check the Major Gridlines on both axes, un-check the Show Legend, and click Finish.

Double-click on the x-axis and format the Scale so that the minimum is -10, the maximum is 10, the major unit is 5, and the minor unit is 1. The "Auto" boxes for those values should all be un-checked. Double-click on the y-axis and format it in the same way.

Now the scroll bar controls the slope of the line and the spinner the y-intercept. Position the scroll bar (remember you can right-click and press Esc to select an control) just below D3, and position the spinner just below F3. Type the letters "a" in D2 and "b" in F2. Make the font in those cells large and colorful.

Check Boxes, Option Buttons, List and Combo Boxes

There are several controls to allow the user to select an option. For example, the user might be specifying a family of functions, a method for manipulating an object, or parameters to be used. Use option buttons, list boxes, or combo boxes when choices are mutually exclusive. Use check boxes when various options can occur together, or if there is only a single option to be toggled on and off..

We'll look at options buttons first.

Creating option buttons

We will put two option buttons above the graph but below the scroll bar and spinner. (Move the graph if needed.)

Click on the Group Box icon (it has "XYZ" above 3 sides of a rectangle) , then drag a rectangle. Click on the text "Group Box" and make the title "Curve type."

Select the Option Button icon, then drag a wide short rectangle completely within the top half of the group box. Change the text to read "Straight line y = ax+b."

Right-click on the option button and select Format Control. Click on the Control tab, make the Value "Checked" and type in the Cell link "H1." Click OK.

Now create a second option button just below the first. Label it "Exponential y = a*e^(bx)."

In cell H2 enter

=if(h1=1,1,0)

(This says, "If the value in H1 is 1, assign the value 1, otherwise assign the value 0.") See what happens in cells H1 and H2 when you toggle between choices of "Curve type."

We want to define the values in column B to correspond to a linear function when "Straight line" is chosen and to an exponential function when "Exponential" is chosen. Define B1 as

=h$2*(d$3*a1+f$3)+(1-h$2)*d$3*exp(f$3*a1)

and black-cross copy that cell into B2:B41.

Try clicking on the second curve type and playing with the slider button and spinner.

When offering more than just two options, you might consider giving "names" to the functions and parameters. (See the "technical issues" in the polar.xls file and the discussion "E. Naming Functions and Variables" in "II. Creating Graphs for Handouts and Exams"--start by following the second link on the left.)

 

List Boxes and Combo Boxes

Type "Linear" in H3 and "Exponential" in H4. Click on th List Box icon and then drag a rectangle next to the "Curve type" group box.

Right-click on your list box and select Format Control. Click on the button at the right of the Input range box, select cells H3 and H4, and close the Format Object window. Type H1 in the Cell link box, and click OK.

Click off the list box, then click on either choice.

Click on the Combo Box icon and then drag another rectangle. Right-click on your combo box select Format Control, and use the same settings as for your list box.

Click off your combo box, click on the down arrow to see the drop-down menu, and make you selection.

 

Check Boxes

Setting up a Check Box is basically the same as setting up a option button. But you typically do not use check boxes in a group box.

 

Macros

A macro is a collection of actions you expect to do many times. Defining a macro mainly involves turning on a recorder, carrying out the actions, and turning off the recorder. Sometimes Excel is configured so that turning on and off the recorder is accompolished by clicking on buttons made to look like tape recorder buttons, but sometimes turning a recording off is accomplished by selecting Tools from the menubar, then Macro->Stop recording.

We will define a macro to associate with the scroll bar that will assign the appropriate formula to cell D3.

First notice that if you type a number in D3, say 10, the scroll bar no longer affects the cell.

Right-click on the scroll bar and select Assign macro. Click on the Record button, click on OK. In cell D3 type

=(d1-50)/10

and press enter. Stop recording (either by pressing the appropriate button or by selecting Tools->Macro->Stop recording).

Warning

The cell references in the macro can be corrupted by inserting or deleting cells in the worksheet. If a working macro goes awry after inserting or deleting cells, you may need to re-record (or edit) the macros. You can delete a macro: select Tools->Macro->Macros, select the appropriate name, then click on Delete.

 

Buttons

You create a button by clicking on the Button icon and dragging a rectangle where you want the button to be. You must assign a macro to the button for it to serve any function.

 

Making it pretty

Select cells A1:B41. Change the font color to white (either using the Font Color icon from the toolbar, or Format->Cells->Font, changing the Color from Automatic to white).

Do the same for cells D1, F1, H1:H4.

Select D3 and make its background yellow (either using the Fill Color icon from the toolbar, or Format->Cells->Patterns and clicking on a yellow square). Do the same in cell F3.

Remove the gridlines and the column and row headings: Click on Tools->Options, and on the View tab un-check "Gridlines" and "Row & column headers."

If you decide to make a nice title for the page, be sure to notice the warning above before you try to insert a new row at the top of the sheet. You can make a wide cell for your text by selecting two adjacent cells and clicking the merge and center icon (next to the right justify button). Or you can create a Text Box over several cells. (See the instructions for making Labels and arrows under "Samples" in II. Creating Graphs for Handouts and Exams .)
