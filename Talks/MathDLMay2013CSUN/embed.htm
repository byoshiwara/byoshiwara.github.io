Embedding MathDL RSS feeds into a webpage or LMS (like Moodle, MyMathLab, Etudes, BlackBoard, WebCT)

  • Find the URL of the feature you want to be updated automatically on your home site. Math in the News, Loci, and On this Day all have their relevant URLs available from http://tinyurl.com/a4z862
  • Find a utility to convert the RSS feed to html code. (Google Gadgets offers a few options.)
  • Run the RSS-to-HTML utility, enter the URL(s), and, when appropriate, choose settings you like. You will get a single (but very long) line of HTML code, which you copy.
  • Edit your Web or LMS page. For most LMS pages, you can simply paste the code or (right-click and) use a "paste embed code" feature.
    If you're using a simple webpage, you'll be pasting HTML code. If you've never looked at HTML code, it might help if you type something distinctive like ZZZZ as a placeholder. Then switch to the HTML editor/view, find your placeholder, and replace it with the long line of code you copied earlier.
  • Save your changes, (upload the edited page if necessary,) and your page should be ready!
