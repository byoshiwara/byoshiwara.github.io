Excel for Math Teachers

AMATYC 29th Annual Conference
Salt Lake City, Utah
November 14, 2003

Bruce Yoshiwara
Los Angeles Pierce College

Program Description: "Participants will practice using the spreadsheet program Excel for keeping class records, for creating graphs to be used in handouts and exams, and for authoring interactive math lessons (with buttons, sliders, etc.). No previous spreadsheet experience will be assumed." (Note: AMATYC was unable to provide a computer lab for the workshop.)

To download the sample Excel and data files used in the workshop, right-click (or shift-click) on this link, save the compressed file to a convenient directory on your computer, and unzip (extract) the files.

To download the web pages used in the workshop, right-click (or shift-click) on this link, save the compressed file to a convenient directory on your computer, and unzip (extract) the files. Then use the Windows Explorer to open the directory where you unzipped the files, and double-click on index.html to start this "workshop" in your browser.
