The graph of \(f(x) = 2^x + 3\text{,}\) shown in FigureÂ 4.30b, has the same basic shape as that of \(h(x) = 2^x\text{,}\) but it has a horizontal asymptote at \(y = 3\) instead of at \(y = 0\) (the \(x\)-axis). In fact, \(f(x) = h(x) + 3\text{,}\) so the graph of \(f\) is a vertical translation of the graph of \(h\) by \(3\) units. If every point on the graph of \(h(x) = 2^x\) is moved \(3\) units upward, the result is the graph of \(f (x) = 2^x + 3\text{.}\)
