Note 5.58 Evaluating \(10^x\)

In ExampleÂ 5.57, the expression \(10^{-3.2}\) can be evaluated in two different ways with a calculator. We can use the ^ key and press

\(10\) ^ (-) \(3.2\) ENTER

which gives 6.30957344 E -4, or approximately \(0.00063\text{.}\) Alternatively, because \(10^x\) is the inverse function for \(\log x\text{,}\) we can press

2nd LOG (-) \(3.2\) ENTER

which gives the same answer as before.
