We find the \(x\)-intercepts of the graph by setting \(y\) equal to zero.
\begin{equation*} \begin{aligned}[t] 0 \amp= 2x^2 + 8x + 6\amp\amp \blert{\text{Factor the right side.}}\\ \amp= 2(x + 1)(x + 3)\amp\amp \blert{\text{Set each factor equal to zero.}}\\ x + 1 \amp=0 ~~~\text{ or } ~~~x + 3 = 0\\ x\amp= -1 \hphantom{blankbl} x = -3 \end{aligned} \end{equation*}
The \(x\)-intercepts are the points \((-1, 0)\) and \((-3, 0)\text{.}\)
