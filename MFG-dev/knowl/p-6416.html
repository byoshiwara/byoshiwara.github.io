  1. The notation \(z = f (x, y)\) indicates that \(z\) is a function of two variables, \(x\) and \(y\text{.}\)

  2. We can use a table with rows and columns to display the output values for a function of two variables.

  3. Joint Variation

    • We say that \(\) varies jointly with \(x\) and \(y\) if

      \begin{equation*} z = kxy,~~~ k\ne 0 \end{equation*}

    • We say that \(\) varies directly with \(x\) and inversely with \(y\) if

      \begin{equation*} z = k\dfrac{x}{y},~~~ k\ne 0, ~~ y\ne 0 \end{equation*}

  4. We can represent a function of two variables graphically by showing a set of graphs for several fixed values of one of the variables.
