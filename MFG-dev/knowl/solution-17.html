Solution 1.24.1.

  1. We find the \(x\)-intercept by setting \(y=0\) and solving for \(x\) to learn that \(x=20\text{,}\) so the \(x\)-intercept is \({\left(20,0\right)}\text{.}\)

    We find the \(y\)-intercept by setting \(x=0\) and solving for \(y\) to learn that \(y=60\text{,}\) so the \(y\)-intercept is \({\left(0,60\right)}\text{.}\)

  2. The \(x\)-intercept has \(y=0\text{,}\) that is, it corresponds to when there are zero radio ads. The \(y\)-intercept has \(x=0\text{,}\) that is, it corresponds to when there are zero tv ads.
