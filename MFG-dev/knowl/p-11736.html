Although a graph is very helpful in solving inequalities, it is not completely necessary. Every quadratic inequality can be put into one of the forms

\begin{equation*} \begin{aligned} ax^2 + bx + c \amp\lt 0\text{,} \amp\hphantom{blank} ax2 + bx + c \amp\gt 0\\ ax^2 + bx + c \amp\le 0\text{,} \amp\hphantom{blank} ax2 + bx + c \amp\ge 0\\ \end{aligned} \end{equation*}
