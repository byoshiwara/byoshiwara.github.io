Solution 4.21.1

We evaluate each function for several convenient values, as shown in Table 4.22.

Then we plot the points for each function and connect them with smooth curves. For positive \(x\)-values, \(g(x)\) is always larger than \(f (x)\text{,}\) and is increasing more rapidly. In Figure 4.23, \(g(x) = 4^x\) climbs more rapidly than \(f (x) = 3^x\text{.}\) Both graphs cross the \(y\)-axis at (0, 1).
