  1. We use logarithms to help us solve exponential equations.
  2. The base \(b\) logarithm of \(x\), written \(\log_b x\text{,}\) is the exponent to which \(b\) must be raised in order to yield \(x\text{.}\)
  3. If \(b\gt 0\) and \(x \gt 0\text{,}\)
    \begin{equation*} y=\log_b x ~~~ \text{ if and only if } ~~~x=b^y \end{equation*}
  4. The operation of taking a base \(b\) logarithm is the inverse operation for raising the base \(b\) to a power.
  5. Base \(10\) logarithms are called common logarithms, and \(\log x\) means \(\log_{10} x\text{.}\)
  6. Steps for Solving Base 10 Exponential Equations.
    1. Isolate the power on one side of the equation.
    2. Rewrite the equation in logarithmic form.
    3. Use a calculator, if necessary, to evaluate the logarithm.
    4. Solve for the variable.
