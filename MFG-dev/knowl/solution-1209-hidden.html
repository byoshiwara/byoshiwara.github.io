The first term is 64, so we have \(a_1=64\text{.}\) To find each subsequent term, we multipy the previous term by \(\dfrac{5}{4}\text{.}\)

\begin{align*} a_2 \amp = \left(\dfrac{5}{4}\right)a_1 = \left(\dfrac{5}{4}\right)(64) = 80\\ a_3 \amp = \left(\dfrac{5}{4}\right)a_2 = \left(\dfrac{5}{4}\right)(80) = 100\\ a_4 \amp = \left(\dfrac{5}{4}\right)a_3 = \left(\dfrac{5}{4}\right)(100) = 125 \end{align*}

The first four terms are 64, 80, 100, and 125.
