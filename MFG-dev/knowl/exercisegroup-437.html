Exercise Group 13–18.

For Problems 13–18,
  1. Graph the set of feasible solutions.
  2. Find the vertex that gives the minimum of the objective function, and find the minimum value.
  3. Find the vertex that gives the maximum of the objective function, and find the maximum value.
13.
Objective function \(C = 3x + 2y\) with constraints
\begin{align*} 2x + y \amp\le 8\\ 4x + 6y \amp\le 24\\ x\ge 0, ~ ~ y \amp\ge 0 \end{align*}
14.
Objective function \(C = -2x + y\) with constraints
\begin{align*} x-2y \amp\ge -10\\ 2x+y \amp\le 10\\ x\ge 0, ~ ~ y \amp\ge 0 \end{align*}
15.
Objective function \(C = 3x - y\) with constraints
\begin{align*} x + y \amp\le 14\\ 5x + y \amp\le 50\\ x\ge 0, ~ ~ y \amp\ge 0 \end{align*}
16.
Objective function \(C = 5x + 4y\) with constraints
\begin{align*} 2x+y \amp\le 10\\ x-3y \amp\ge -3\\ x\ge 0, ~ ~ y \amp\ge 0 \end{align*}
17.
Objective function \(C = 200x - 20y\) with constraints
\begin{align*} 3x + 2y \amp\le 24\\ x + y \amp \le 9\\ x + 2y \amp\le 16\\ x\ge 0, ~ ~ y \amp\ge 0 \end{align*}
18.

\begin{align*} 3x + 2y \amp\le 24\\ 3x-y \amp\le 15\\ 3x - 4y \amp\ge -12\\ x\ge 0, ~ ~ y \amp\ge 0 \end{align*}
