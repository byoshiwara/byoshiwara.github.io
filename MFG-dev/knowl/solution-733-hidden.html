The first term is 6, so we have \(a_1 = 6\text{.}\) To find each subsequent term, we add 3 to the previous term:

\begin{equation*} \begin{aligned}[t] a_2 \amp = a_1 + 3 = 6 + 3 = 9\\ a_3 \amp = a_2 + 3 = 9 + 3 = 12\\ a_4 \amp = a_3 + 3 = 12 + 3 = 15\\ \end{aligned} \end{equation*}

The first four terms are 6, 9, 12, and 15.
