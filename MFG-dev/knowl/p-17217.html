In Example 7.72 of Section 7.4, we wrote a function that gave the time Francine needs for a 60-mile training run on her cycle-plane in terms of the windspeed, \(x\text{:}\)

\begin{equation*} t=f (x) = \frac{60}{15-x} \end{equation*}

If it takes Francine \(9\) hours to cover \(60\) miles, what is the speed of the wind? We can answer this question by reading values from the graph of \(f\text{,}\) as shown at right. When \(t = 9\text{,}\) the value of \(x\) is between \(8\) and \(9\text{,}\) so the windspeed is between \(8\) and \(9\) miles per hour.
