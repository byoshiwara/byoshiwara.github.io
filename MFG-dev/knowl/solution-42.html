Solution2.8.1

Because the variable we want is squared, we use extraction of roots. First, we multiply both sides by \(3\) to clear the fraction.

\begin{equation*} \begin{aligned} \alert{3}V \amp = \alert{3}(\frac{1}{3} πr^2h)\\ 3V \amp = πr^2h \amp\amp \text{Divide both sides by } πh.\\ \frac{3V}{πh}\amp= r^2 \amp\amp \text{Take square roots.}\\ \pm\sqrt{\dfrac{3V}{πh}}\amp= r \end{aligned} \end{equation*}

Because the radius of a cone must be a positive number, we use only the positive square root: \(r = \sqrt{\dfrac{3V}{πh}}\text{.}\)
