Paragraph

  1. The domain of a function is the set of permissible values for the input variable.

  2. The range is the set of function values (that is, values of the output variable) that correspond to the domain values.

  3. A relationship between two variables is a function if each element of the domain is paired with only one element of the range.

  4. We can identify the domain and range of a function from its graph. The domain is the set of \(x\)-values of all points on the graph, and the range is the set of \(y\)-values.

  5. If the domain of a function is not given as part of its definition, we assume that the domain is as large as possible.

  6. In applications, we may restrict the domain and range of a function to suit the situation at hand.
