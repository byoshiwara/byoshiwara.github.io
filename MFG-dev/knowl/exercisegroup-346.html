Exercise Group 57–60

For Problems 57–60,

  1. Find all values of \(x\) for which \(f(x) = 0\text{.}\)

  2. Find all values of \(x\) for which \(g(x) = 0\text{.}\)

  3. Find all values of \(x\) for which \(f(x) = g(x) \text{.}\)

  4. Graph each pair of functions in the same window, then sketch the graph on paper. Illustrate your answers to (a)–(c) as points on the graph.

57

\(f (x) = 2x^2 + 3x\text{,}\) \(~g(x) = 5 - 6x\)

58

59

\(f (x) = 2x^2 -2x\text{,}\) \(~g(x) = x^2+3\)
