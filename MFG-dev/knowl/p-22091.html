If \(C_n\) represents the number of contestants after \(n\) rounds of competition, then

\begin{align*} C_1 \amp = 1024 \left(\dfrac{1}{2}\right) = 512\\ C_2 \amp = 512 \left(\dfrac{1}{2}\right) = 1024 \left(\dfrac{1}{2}\right)^2 = 256\\ C_3 \amp = 256 \left(\dfrac{1}{2}\right) = 1024 \left(\dfrac{1}{2}\right)^3 = 128\ \end{align*}

In general

\begin{gather*} C_n = 1024 \left(\dfrac{1}{2}\right)^n \end{gather*}
