Solution6.43.1

We first write the equation in standard form as a quadratic equation in the variable \(x\text{.}\)

\begin{equation*} x^2 - yx + (y - 2) = 0 \end{equation*}

Expressions in \(y\) are treated as constants with respect to \(x\text{,}\) so that \(a = \alert{1}\text{,}\) \(b = \alert{-y}\text{,}\) and \(c = \alert{y - 2}\text{.}\) Substitute these expressions into the quadratic formula. \begin{align*} x \amp=\frac{-(\alert{-y}) \pm\sqrt{(\alert{-y})^2 - 4(\alert{1})(\alert{y - 2})}}{2(\alert{1})}\\ \amp = \frac{y \pm\sqrt{y^2 - 4y + 8}}{2} \end{align*}
