We write the function as \(y = 2^{x-3} - 4\text{,}\) and solve for \(x\) in terms of \(y\text{.}\) First, we isolate the power: \begin{align*} y + 4 \amp= 2^{x-3}\amp\amp\text{Take logs base 2.}\\ \log_{2}{(y + 4)} \amp= \log_{2}{2x-3} = x - 3\\ x \amp = 3 + \log_{2}{(y + 4)} \end{align*} The inverse function is \(f^{-1}(y) = 3 + \log_{2}{(y + 4)}\text{.}\) However, to graph both \(f\) and \(f^{-1}\) on the same grid, we write the inverse function as \(f^{-1}(x) = 3 + \log_{2}{(x + 4)}\text{.}\)
