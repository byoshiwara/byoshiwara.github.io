Solution 4.48.1.

  1. To evaluate \(\log_3 9\text{,}\) we ask what exponent on base \(3\) will produce \(9\text{.}\) Or, 3 to what power equals 9? The exponent we need is \(\alert{2}\text{,}\) so

    \begin{align*} \log_3 9 = \alert{2}\amp\amp\amp \text{because} \amp\amp 3^{\alert{2}} = 9 \end{align*}

    We use similar reasoning to compute the other logarithms.

  2. \(\displaystyle \log_5{125} = \alert{3}~~ \text{ because } ~~5^{\alert{3}} = 125\)

  3. \(\displaystyle \log_4{\dfrac{1}{16}}= \alert{-2} ~~\text{ because } ~~4^{\alert{-2}} = \dfrac{1}{16}\)

  4. \(\displaystyle \log_5{\sqrt{5}} = \alert{\dfrac{1}{2}}~~ \text{ because }~~ 5^{\alert{1/2}} =\sqrt{5}\)
