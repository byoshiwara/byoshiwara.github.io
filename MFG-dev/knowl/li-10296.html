Item 23.

To Solve a Linear Programming Problem:.

  1. Represent the unknown quantities by variables. Write the objective function and the constraints in terms of the variables.
  2. Graph the solutions to the constraint inequalities.
  3. Find the coordinates of each vertex of the solution set.
  4. Evaluate the objective function at each vertex.
  5. The maximum and minimum values of the objective function occur at vertices of the set of feasible solutions.
