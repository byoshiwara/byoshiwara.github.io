Solution 8.70.1

First, we graph the equation \(x = 2\text{;}\) its graph is a vertical line. Since the origin does not lie on this line, we can use it as a test point. Substitute \(0\) for \(x\) (there is no \(y\)) into the inequality to obtain

\begin{equation*} 0\ge 2 \end{equation*}

Since this is a false statement, shade the half-plane that does not contain the origin. We see in FigureÂ 8.71 that the graph of the inequality contains all points whose \(x\)-coordinates are greater than or equal to \(2\text{.}\)
