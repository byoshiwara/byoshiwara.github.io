Item a.
We apply the zero-factor principle to the product \((x - 6)(x + 2)\text{.}\)
\begin{equation*} \begin{aligned}[t] (x - 6)(x+ 2) \amp = 0 \amp\amp \blert{\text{Set each factor equal to zero.}}\\ x - 6 = 0~~ \amp\text{or} ~~ x+2=0 \amp\amp \blert{\text{Solve each equation.}}\\ ~~~x=6~~ \amp\text{or} ~~ x=-2 \end{aligned} \end{equation*}
There are two solutions, \(6\) and \(-2\text{.}\) (You should check that both of these values satisfy the original equation.)
