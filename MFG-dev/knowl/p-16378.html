  1. If \(P(x)\) is a nonconstant polynomial with real coefficients and \(a\) is any real number, then there exist unique polynomials \(q(x)\) and \(r (x)\) such that

    \begin{equation*} P(x) = (x - a)q(x) + r (x) \end{equation*}

    where \(\text{deg }r (x)\lt \text{deg }(x - a)\text{.}\)

  2. Zero

  3. \(P(a) = (a - a)q(a) + r (a) = r (a)\text{.}\) Because \(\text{deg }r(x) = 0 \text{,}\) \(r(x) \) is a constant. That constant value is \(P(a) \text{,}\) so \(P(x) = (x - a)q(x) + P(a)\text{.}\)
