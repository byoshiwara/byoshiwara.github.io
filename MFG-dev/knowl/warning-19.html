Caution 5.9.

Although the same symbol, \(^{-1}\text{,}\) is used for both reciprocals and inverse functions, the two notions are not equivalent. That is, the inverse of a given function is usually not the same as the reciprocal of that function. In ExampleÂ 5.4, note that \(f^{-1}(y)\) is not the same as the reciprocal of \(f (y)\text{,}\) because

\begin{equation*} \frac{1}{f(y)}= \frac{1}{y^3+ 2} ~~\text{ but }~~ f^{-1}(y)= \sqrt[3]{y - 2} \end{equation*}

To avoid confusion, we use the notation \(\dfrac{1}{f}\) to refer to the reciprocal of the function \(f\text{.}\)
