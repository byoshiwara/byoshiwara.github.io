We have formulated the original problem into an objective function

\begin{equation*} P = 8x + 10y \end{equation*}

and a system of inequalities called the constraints.

\begin{align*} 3x + 6y \amp\le 2400\\ 2x + y \amp\le 1000\\ x \ge 0, ~~\amp y \ge 0 \end{align*}

Our goal is to find values for \(x\) and \(y\) that satisfy the constraints and produce the maximum value for \(P\text{.}\)
