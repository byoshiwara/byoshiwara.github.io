  1. Write a formula for the number of phishing Web sites as a function of months since June \(2005\text{.}\)
  2. How long will it take for the number of sites to reach \(8000\text{?}\) Give both an exact value and a decimal approximation rounded to two decimal places.
  3. How long should it take for the number of sites to reach \(16,000\text{?}\)
  4. Using your answers to (b) and (c), make a rough sketch of the function.
