Solution 4.98.1

We would like to find the present value of an annuity in which \(P = 400\text{,}\) \(r = 0.04\text{,}\) \(n = 12\text{,}\) and \(t = 5\text{.}\) Substituting these values into the formula gives \begin{align*} PV \amp = \frac{400\left[1-\left(1+\frac{0.04}{12}\right)^{-(12)(5)}\right]}{\frac{0.04}{12}}\\ \amp= \frac{400[1 - (1.00\overline{3})]^{-60}}{0.00\overline{3}}\\ \amp= 21,719.63 \end{align*} Delbert's Aunt Welthy should deposit $\(21,719.63\text{.}\)
