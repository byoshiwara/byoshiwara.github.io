Item 7.80.a.

Find the domain of \(F(x) = \displaystyle{\frac{x - 2}{x + 4}}\text{.}\)

You may use the inequality symbols (\(\lt\text{,}\) \(\le\text{,}\) \(>\text{,}\) \(\ge\)) or enter your answers using interval notation. When using interval notation, use "inf" for \(\infty\) and use "U" (an upper case letter u) for the union symbol \(\cup\text{.}\)
