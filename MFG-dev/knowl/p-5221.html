In a lab assignment, a student reports that a chemical reaction required \(200\) minutes to complete. Let \(t\) represent the actual time of the reaction.
  1. Write an absolute value inequality for \(t\text{,}\) assuming that the student rounded his answer to the nearest \(100\) minutes. Give the smallest and largest possible value for \(t\text{.}\)
  2. Write an absolute value inequality for \(t\text{,}\) assuming that the student rounded his answer to the nearest minute. Give the smallest and largest possible value for \(t\text{.}\)
  3. Write an absolute value inequality for \(t\text{,}\) assuming that the student rounded his answer to the nearest \(0.1\) minute. Give the smallest and largest possible value for \(t\text{.}\)
