We substitute \(x = \alert{1 + i}\) to find

\begin{align*} f(\alert{1 + i} ) \amp = (\alert{1 + i} )^2 - 2(\alert{1 + i} ) + 2\\ \amp = 1^2 + 2i + i^2 - 2 - 2i + 2\\ \amp = 1 + 2i + (-1) - 2 - 2i + 2\\ \amp =0 \end{align*}

Thus, \(f(1 + i ) = 0\text{,}\) so \(1 + i\) is a solution of \(x^2 - 2x + 2 = 0\text{.}\)
