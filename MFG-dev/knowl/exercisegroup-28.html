Exercise Group 17–20.

For Problems 17–20, use the formula \(m=\dfrac{\Delta y}{\Delta x} \)
17.
A line has slope \(\dfrac{-3}{4}\text{.}\)
  1. Find the vertical change associated with each horizontal change along the line.
    1. \(\displaystyle \Delta x = 4\)
    2. \(\displaystyle \Delta x = -8\)
    3. \(\displaystyle \Delta x = 2\)
    4. \(\displaystyle \Delta x = -6\)
  2. Find the horizontal change associated with each vertical change along the line.
    1. \(\displaystyle \Delta y = 3\)
    2. \(\displaystyle \Delta y = -6\)
    3. \(\displaystyle \Delta y = -2\)
    4. \(\displaystyle \Delta y = 1\)
18.
A line has slope \(\dfrac{5}{3}\text{.}\)
  1. Find the vertical change associated with each horizontal change along the line.
    1. \(\displaystyle \Delta x = 3\)
    2. \(\displaystyle \Delta x = -6\)
    3. \(\displaystyle \Delta x = 1\)
    4. \(\displaystyle \Delta x = -24\)
  2. Find the horizontal change associated with each vertical change along the line.
    1. \(\displaystyle \Delta y = -5\)
    2. \(\displaystyle \Delta y = -2.5\)
    3. \(\displaystyle \Delta y = -1\)
    4. \(\displaystyle \Delta y = 3\)
19.
Residential staircases are usually built with a slope of \(70\%\text{,}\) or \(\dfrac{7}{10}\text{.}\) If the vertical distance between stories is \(10\) feet, how much horizontal space does the staircase require?
20.
A straight section of highway in the Midwest maintains a grade (slope) of \(4\%\text{,}\) or \(\dfrac{1}{25}\text{,}\) for \(12\) miles. How much does your elevation change as you travel the road?
