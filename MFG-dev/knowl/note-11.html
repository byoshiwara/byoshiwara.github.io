Note 1.89

The slope of a line is a ratio and can be written in many equivalent ways. In ExampleÂ 1.88, the slope is equal to \(\dfrac{8}{6}\text{,}\) \(\dfrac{12}{9}\text{,}\) and \(\dfrac{-4}{-3}\text{.}\) We can use any of these fractions to locate a third point on the line as a check. If we use \(m = \dfrac{\Delta y}{\Delta x}= \dfrac{-4}{-3}\text{,}\) we move down \(4\) units and left \(3\) units from the \(y\)-intercept to find the point \((-3, -6)\) on the line.
