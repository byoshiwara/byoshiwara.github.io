SubsectionFunctions of Two or More Variables

So far, we have studied functions that relate values of an output variable to values of a single input variable. But it is not uncommon for an output variable to depend on two or more inputs. Many familiar formulas describe functions of several variables.

For example, the perimeter of a rectangle depends on its length and width. The volume of a cylinder depends on its radius and height. The distance you travel depends on your speed and the time you spent traveling. Each of these formulas can be written with function notation. \begin{align*} \amp P = f (l,w) = 2l + 2w \amp\amp \text{Perimeter is a function of length and width.}\\ \amp V = f (r, h) = \pi r^2h \amp\amp \text{Volume is a function of radius and height.}\\ \amp d = f (r, t) = rt \amp\amp \text{Distance is a function of rate and time.} \end{align*}

Example3.94

The cost, \(C\text{,}\) of driving a rental car is given by the function

\begin{equation*} C = f (t, g) = 29.95t + 2.80g \end{equation*}

where \(t\) is the number of days you rent the car and \(g\) is the number of gallons of gas you buy.

  1. Evaluate \(f (3,10)\) and explain what it means.

  2. You have $100 to rent a car for \(2\) days. How much gas can you buy?

Solution

  1. We substitute \(3\) for \(t\) and \(10\) for \(g\) to find

    \begin{equation*} f (3, 10) = 29.95(3) + 2.80(10) = 117.85 \end{equation*}

    It will cost $\(117.85\) to rent a car for \(3\) days and buy \(10\) gallons of gas.

  2. We would like to find the value of \(g\) when \(t = 2\) and \(C = 100\text{.}\) That is, we want to solve the equation \begin{align*} 100 \amp = f (2, g) = 29.95(2) + 2.80g \\ 100 \amp = 59.90 + 2.80g \\ g \amp = 14.32 \end{align*} You can buy \(14.32\) gallons of gas.

Exercise3.95

where \(P\) is the nozzle pressure in psi, and \(n\) is measured in \(18\)-inch increments over the standard nozzle diameter of \(\dfrac{3}{4}\) inch.

SubsectionTables of Values

Just as for functions of a single variable, we can use tables to describe functions of two variables, \(z = f (x, y)\text{.}\) The row and column headings show the values of the two input variables, and the table entries show the values of the output variable.

Sometimes we can find patterns relating the entries in a table.

Rectangular beams of a given length can support a load, \(L\text{,}\) that depends on both the width and the depth of the beam, so that \(L = f (w, d)\text{.}\) The table shows some of the values.

  2. Is it true that \(f (2, 5) = f (5, 2)\text{?}\)

  1. In the row for \(2\) cm and the column for \(5\) cm, we find that \(f (2, 5) = 500\text{.}\) A beam of width \(2\) cm and depth \(5\) cm can support a maximum load of \(500\) kilograms.

Exercise3.100
  1. For the table in Example 3.98, consider the column corresponding to a beam depth of \(3\) cm. Graph \(L\) as a function of \(w\) when the depth is constant at \(d = 3\text{.}\)
  2. Find a formula for \(L\) as a function of \(w\) for \(d = 3\text{.}\)

In Exercise 3.100, you should find that the load varies directly with width when the depth is \(3\) centimeters. In fact, the load varies directly with width for any fixed depth.

In Example 3.98, we saw that the load varies with the square of depth when the width is \(3\) centimeters, and this relationship also holds for any value of \(w\text{.}\) Consequently, we can find a constant \(k\) such that

\begin{equation*} \text{load} = k \cdot \text{width} \cdot \text{depth}^2 \end{equation*}

This relationship between variables is an example of joint variation.

Joint Variation

If

\begin{equation*} z = kxy \text{, } ~~ k \gt 0 \end{equation*}

we say that \(z\) varies jointly with \(x\) and \(y\text{.}\) If

\begin{equation*} z = k\frac{x}{y}\text{, } ~~ k \gt 0\text{, } ~~ y \ne 0 \end{equation*}

we say that \(z\) varies directly with \(x\) and inversely with \(y\text{.}\)

Example3.101

Find a formula for load as a function of width and depth for the data in Example 3.98.

Solution

The function we want has the form

\begin{equation*} L = f (w, d) = kwd^2 \end{equation*}

for some value of \(k\text{.}\) We use the fact that \(L = 10\) when \(w = 1\) and \(d = 1\text{.}\) Then

\begin{equation*} 10 = k(1)(1^2) \end{equation*}

so \(k = 10\text{.}\) The formula for load as a function of width and depth is

\begin{equation*} L = 10wd^2 \end{equation*}

You can check that this formula works for all the values in the table.

SubsectionGraphs

It is possible to make graphs in three dimensions for functions of two variables, but we will not do that here. Instead, we will represent such functions graphically by holding one of the two variables constant.

Example3.103

In Example 3.101, we found a formula for the load a beam can support,

\begin{equation*} L = 10wd^2 \end{equation*}

  1. Graph \(L\) as a function of \(w\) for \(d = 1, 2, 3,\) and \(4\text{.}\)

  2. Graph \(L\) as a function of \(d\) for \(w = 1, 2, 3,\) and \(4\text{.}\)

Solution

  1. We make four graphs on the same grid, one for each value of \(d\text{:}\) \begin{align*} \text{when } d \amp= 1, \amp\amp L = 10w \\ \text{when } d \amp= 2, \amp\amp L = 40w \\ \text{when } d \amp= 3, \amp\amp L = 90w \\ \text{when } d \amp= 4, \amp\amp L = 160w \end{align*} The graph is shown in Figure 3.104. We can see that \(L\) varies directly with the width of the beam for any fixed value of its depth.

  2. We make one graph for each value of \(w\text{:}\) \begin{align*} \text{when } w \amp= 1, \amp\amp L = 10d^2 \\ \text{when } w \amp= 2, \amp\amp L = 20d^2 \\ \text{when } w \amp= 3, \amp\amp L = 30d^2 \\ \text{when } w \amp= 4, \amp\amp L = 40d^2 \end{align*} The graph is shown in Figure 3.105. For any fixed value of its width, \(L\) varies directly with the square of depth.

SubsubsectionCONCEPTS

  1. The notation \(z = f (x, y)\) indicates that \(z\) is a function of two variables, \(x\) and \(y\text{.}\)

  2. We can use a table with rows and columns to display the output values for a function of two variables.

  3. Joint Variation

    • We say that \(\) varies jointly with \(x\) and \(y\) if

      \begin{equation*} z = kxy,~~~ k\ne 0 \end{equation*}

    • We say that \(\) varies directly with \(x\) and inversely with \(y\) if

      \begin{equation*} z = k\dfrac{x}{y},~~~ k\ne 0, ~~ y\ne 0 \end{equation*}

  4. We can represent a function of two variables graphically by showing a set of graphs for several fixed values of one of the variables.
