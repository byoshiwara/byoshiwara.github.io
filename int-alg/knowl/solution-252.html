Solution 7.3.7.1.

First identify the base \(b\text{,}\) and then the exponent or logarithm \(y\text{.}\) Use the conversion equations to rewrite \(b^y = x\) in the form \(\log_{b}{(x)} = y\text{.}\)
  1. The base is \(\blert{2}\) and the exponent is \(\alert{-1}.~~\) Thus, \(\log_{\blert{2}}{\left(\dfrac{1}{2}\right)}= \alert{-1}\text{.}\)
  2. The base is \(\blert{a}\) and the exponent is \(\alert{\dfrac{1}{5}}.~~\) Thus, \(\log_{\blert{a}}{(2.8)} = \alert{\dfrac{1}{5}}\text{.}\)
  3. The base is \(\blert{6}\) and the exponent is \(\alert{1.5}.~~\) Thus, \(\log_{\blert{6}}{(T)} = \alert{1.5}\text{.}\)
  4. The base is \(\blert{M}\) and the exponent is \(\alert{v}.~~\) Thus, \(\log_{\blert{M}}{(3K)} = \alert{v}\text{.}\)
