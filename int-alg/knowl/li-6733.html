Item 7.77.d.

\(\begin{aligned}[t] \log x +\log {(x-2)}\amp=3 \\ \color{gray}{x + (x-2)}\amp=\color{gray}{10^3}\amp\amp \color{green}{\text{(Wrong!)}} \end{aligned}\)

Instead, the first step should be

  • \(\log 2x-2=3\)

  • \(\log{(x(x-2))} = 3\)

  • \(\log x + \log x - \log 2 = 3\)
