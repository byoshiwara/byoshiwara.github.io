Example 6.51.

  1. \(\displaystyle 81^{3/4} = \left(81^{1/4}\right)^3 = 3^3 = 27\)

  2. \(\displaystyle -27^{5/3} = -\left(27^{1/3}\right)^5 = -3^5 = -243\)

  3. \(\displaystyle 27^{-2/3} = \frac{1}{\left(27^{1/3}\right)^2} = \frac{1}{3^2}= \frac{1}{9}\)

  4. \(\displaystyle 5^{3/2} = \left(5^{1/2}\right)^3 \approx (2.236)^3 \approx 11.180\)
