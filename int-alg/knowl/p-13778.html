in-context =======

GeoGebra nicely displays the vertical bars of the standard absolute value notation, such as in the expression \(\abs{21\cdot 54 - 81} \text{.}\) Once we select the absolute value key, the cursor is in between the vertical bars and will stay there until we use the arrow keys to move the cursor outside, or until we press ⮠. The arrow keys on the virtual keyboard are shown in the keyboard image A.1.
