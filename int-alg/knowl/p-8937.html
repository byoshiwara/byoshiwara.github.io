\(\displaystyle f(x)=2x^{\frac{3}{4}}\)

in-context =======

Compute the logarithms.

  1. \(\log_3 9 = 2\) because \(3^2=9\)

  2. \(\log_5 125 = 3\) because \(5^3=125\)

  3. \(\log_4 \dfrac{1}{16} = -2\) because \(4^{-2}=\dfrac{1}{16}\)

  4. \(\log_5 \sqrt{5} = \dfrac{1}{2}\) because \(5^{1/2} = \sqrt{5}\)
