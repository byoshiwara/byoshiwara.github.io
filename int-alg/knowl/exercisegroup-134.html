Exercise Group 25–26.

For Problems 25 and 26,

  1. Compute \(f(0)\) and \(g(0)\text{.}\)
  2. Find all values of \(x\) for which \(f(x)=0\text{.}\)
  3. Find all values of \(x\) for which \(g(x)=0\text{.}\)
  4. Find all values of \(x\) for which \(f(x)=g(x)\text{.}\)
  5. Graph \(f\) and \(g\) on the same axes, and illustrate your answers to (a)–(d) as points on the graph.
25.

\(f(x) = 2x^2+3x, \quad g(x) = 5-6x\)

26.

\(f(x) = 2x^2-2x, \quad g(x) = x^2+3\)
