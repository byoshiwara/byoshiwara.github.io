We can choose any point for the test point as long as it does not lie on the boundary line. We chose \((0, 0)\) in Example 2.5.8 because the coordinates are easy to substitute into the inequality. If the test point is a solution, we shade the half-plane including that point. If the test point is not a solution, we shade the other half-plane. For example, suppose we had chosen \((5, 0)\) as the test point in Example 2.5.8. When we substitute its coordinates into the inequality, we find
\begin{gather*} 3(\alert{5}) - 2(\blert{0}) \lt 6 \end{gather*}
which is a false statement. Thus, \((5, 0)\) is not a solution to the inequality, so the solutions must lie on the other side of the boundary line. Note that using \((5, 0)\) as the test point gives us the same solutions we found in Example 2.5.8.
