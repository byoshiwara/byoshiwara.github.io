  1. The closed interval \([a,b]\) is the set \(a \le x \le b\text{.}\)
  2. The open interval \((a,b)\) is the set \(a \lt x \lt b\text{.}\)
  3. Intervals may also be half-open or half-closed.
  4. The infinite interval \([a,\infty)\) is the set \(x \ge a\text{.}\)
  5. The infinite interval \((-\infty,a]\) is the set \(x \le a\text{.}\)
