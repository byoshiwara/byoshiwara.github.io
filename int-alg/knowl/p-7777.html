Using exponents

  1. Simplify \((\sqrt[3]{8})^2\) using exponents as follows. Fill in the blanks with the correct exponents.

    \begin{equation*} (\sqrt[3]{8})^2 = (8^{\underline{\hspace{0.909090909090909em}}})^2 = 8^{\underline{\hspace{0.909090909090909em}}} \end{equation*}

  2. Use your calculator to compute \(8^{2/3}\text{.}\) Don't forget to put parentheses around the exponent.

  3. Approximate each power to the nearest thousandth.

    1. \(\displaystyle 12^{5/6}\)

    2. \(\displaystyle \sqrt[3]{6^4}\)

    3. \(\displaystyle 37^{-2/3}\)

    4. \(\displaystyle 4.7^{2.3}\)
