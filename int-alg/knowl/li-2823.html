Item 5.6.b.
\(x\) \(-3\) \(-2\) \(-1\) \(0\) \(1\) \(2\) \(3\)
\(y\) \(17\) \(3\) \(0\) \(-1\) \(0\) \(3\) \(17\)

Is \(y\) a function of \(x\text{?}\)

  • Yes

  • No

  • Each value of \(x\) has exactly one value of \(y\) associated with it.

  • For example, \(y=3\) corresponds both to \(x=-2\) and also to \(x=2\)
