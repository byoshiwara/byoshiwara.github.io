Hyperbolas.

The equation for a hyperbola centered at \((h,k)\) has one of the two standard forms:

\begin{gather*} \blert{\frac{(x-h)^2}{a^2} - \frac{(y-k)^2}{b^2} = 1}\\ \blert{\frac{(y-k)^2}{b^2} - \frac{(x-h)^2}{a^2} = 1} \end{gather*}

Conic Sections.

The graph of \(~Ax^2 + Cy^2 + Dx + Ey + F = 0~\) is

  1. a circle if \(A=C\text{.}\)

  2. a parabola if \(A=0\) or \(C=0\) (but not both).

  3. an ellipse if \(A\) and \(C\) have the same sign.

  4. a hyperbola if \(A\) and \(C\) have opposite signs.
