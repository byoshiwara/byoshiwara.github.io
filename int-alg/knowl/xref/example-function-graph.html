Example 5.2.4.

The figure shows the graph of a function \(g\text{.}\)
  1. Find \(g(-2)\) and \(g(5)\text{.}\) \(~\alert{\text{[TK]}}\)
  2. For what value(s) of \(t\) is \(g(t) = -2\text{?}\)
  3. What is the largest, or maximum, value of \(g(t)\text{?}\) For what value of \(t\) does the function take on its maximum value?
  4. On what intervals is \(g\) increasing?

Solution.

  1. To find \(g(-2)\text{,}\) we look for the point with \(t\)-coordinate \(-2\text{.}\) The point \((-2, 0)\) lies on the graph of \(g\text{,}\) so \(g(-2) = 0\text{.}\) Similarly, the point \((5, 1)\) lies on the graph, so \(g(5) = 1\text{.}\)
  2. We look for points on the graph with \(y\)-coordinate \(-2\text{.}\) Because the points \((-5, -2)\text{,}\) \((-3, -2)\text{,}\) and \((3, -2)\) lie on the graph, we know that \(g(-5) = -2\text{,}\) \(g(-3) = -2\text{,}\) and \(g(3) = -2\text{.}\) Thus, the \(t\)-values we want are \(-5\text{,}\) \(-3\text{,}\) and \(3\text{.}\)
  3. The highest point on the graph is \((1, 4)\text{,}\) so the largest \(y\)-value is \(4\text{.}\) Thus, the maximum value of \(g(t)\) is \(4\text{,}\) and it occurs when \(t = 1\text{.}\)
  4. A graph is increasing if the \(y\)-values get larger as we read from left to right. The graph of \(g\) is increasing for \(t\)-values between \(-4\) and \(1\text{,}\) and between \(3\) and \(5\text{.}\) Thus, \(g\) is increasing on the intervals \((-4, 1)\) and \((3, 5)\text{.}\)
