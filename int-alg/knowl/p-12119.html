  3. Each time we multiply \(x\) by 5, the log of \(x\) increases by 0.7, because \(\log {(5)} = 0.7\text{.}\) This is an application of the log properties:
    \begin{equation*} \log {(5x)} = \log {(x)} + \log {(5)} = \log {(x)} + 0.7 \end{equation*}
