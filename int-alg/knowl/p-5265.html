A farmer inherits an apple orchard on which 60 trees are planted per acre. Each tree yields 12 bushels of apples. Experimentation has shown that for each tree removed per acre, the yield per tree increases by \(\dfrac{1}{2}\) bushel.

  1. Write algebraic expressions for the number of trees per acre and for the yield per tree if \(x\) trees per acre are removed.
  2. Write a quadratic equation for the total yield per acre if \(x\) trees are removed per acre.
  3. What is the maximum yield per acre that can be achieved by removing trees? How many trees per acre should be removed to achieve this yield?
  4. How many trees should the farmer remove per acre in order to harvest at least 850 bushels per acre?
  5. Graph your equation for total yield in the window
    \begin{align*} \text{Xmin} \amp = 0 \amp\amp \text{Xmax} = 94\\ \text{Ymin} \amp = 0 \amp\amp \text{Ymax} = 1000 \end{align*}
    and use your graph to verify your answers to parts (e) and (f).
