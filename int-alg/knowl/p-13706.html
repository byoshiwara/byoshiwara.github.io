We use the input box next to the large "plus" symbol to enter expressions we wish to evaluate.

  • When using the app on a computer, we can type with the physical keyboard or use the app's virtual keyboard. (If the virtual keyboard is not initially displayed, click on the keyboard icon in the lower left corner of the Algebra pane.)
  • When using the app on a small screen, tap in the input box and the virtual keyboard will be visible.
