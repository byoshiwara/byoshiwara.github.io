We can also find the input (or inputs) corresponding to a given output. For example, if \(p=g(w)\) is the postage function, we solve the equation \(g(w)=0.68\) by finding all input values, \(w\text{,}\) that correspond to the output $0.68. According to the table in Example 2b, any value of \(w\) greater than 1 but less than or equal to 2 is a solution.
