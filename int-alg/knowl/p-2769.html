The graph of the inequality must show all the points whose coordinates are solutions. We found some solutions to the inequality \(x + y \ge 10,000\) in the Example above by choosing a value for \(x\) and solving for \(y\text{.}\) A more efficient way to find all the solutions of the inequality is to start with the graph of the corresponding equation

\begin{gather*} y = -x + 10,000 \end{gather*}
