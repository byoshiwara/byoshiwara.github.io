For this decay law, \(N_0 = 60\) and \(b = 0.8\text{.}\) We would like to find a value for \(k\) so that \(e^k = b = 0.8\text{,}\) that is, we must solve the equation
\begin{align*} e^k \amp = 0.8\amp\amp \blert{\text{Take natural log of both sides.}}\\ \ln {(e^k)}\amp = \ln {(0.8)}\amp\amp \blert{\text{Simplify.}}\\ k\amp = \ln {(0.8)} \approx -0.2231 \end{align*}
Replacing \(b\) with \(e^k\text{,}\) we find that the decay law is
\begin{equation*} N(t) \approx 60e^{-0.2231t} \end{equation*}
