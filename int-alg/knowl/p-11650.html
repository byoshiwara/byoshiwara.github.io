True or False

  1. We can always find the LCD by multiplying together the denominators of the fractions.
  2. If each denominator includes one factor of \(x-3\text{,}\) we include two factors of \(x-3\) in the LCD.
  3. We find the building factor for each fraction by comparing its denominator to the LCD.
  4. When adding fractions, it is best to reduce the fractions at each step.
