GeoGebra nicely displays powers such as \(10^7 \text{,}\) with the exponent raised like a superscript. Once we enter the base and select the exponent key, the cursor is in the exponent and will stay there until we use the arrow keys to move the cursor outside the power, or until we press ⮠. The arrow keys on the virtual keyboard are shown in the keyboard image A.1.
