Exercise 15.
The manager for Books for Cooks plans to spend $300 stocking a new diet cookbook. The paperback version costs her $5, and the hardback costs $10. She finds that she will sell three times as many paperbacks as hardbacks. How many of each should she buy?
  1. Let \(x\) represent the number of hardbacks and \(y\) the number of paperbacks she should buy. Write an equation about the cost of the books.
  2. Write a second equation about the number of each type of book.
  3. Graph both equations and solve the system using the grid.
  4. Answer the question in the problem.
