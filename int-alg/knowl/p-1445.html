We can graph this system in the standard window by solving each equation for \(y\text{.}\) We enter
\begin{align*} Y_1\amp = (21.06 - 3X)/-2.8\\ Y_2\amp = (5.3 - 2X)/1.2 \end{align*}
and use the standard window. If we trace along the graph to the intersection point, we will not find the same coordinates on both lines. The intersection point is not displayed in this window. Instead, we can use the intersect feature of the grapher.
