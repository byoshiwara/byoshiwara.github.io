We can factor the expression \(x^2-4x-12\text{,}\) and write the equation as

\begin{equation*} (x-6)(x+2) = 0 \end{equation*}

Now it is in the form \(~ab=0~\text{,}\) with \(~a=x-6~\) and \(~b=x+2~\text{,}\) so the ZFP tells us that either \(~x-6=0~\) or \(~x+2=0~\text{.}\) We solve each of these equations.

\begin{align*} x-6 = 0~~~~\amp \text{or}~~~~x+2 = 0 \amp \amp \glert{\text{Solve each equation.}}\\ x=6~~~~ \amp \text{or}~~~~x=-2 \end{align*}

Once again we see that a quadratic equation has two solutions. You can check that both of these values satisfy the original equation.
