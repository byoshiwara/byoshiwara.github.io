Converting between radical notation and exponent notation

  1. Write with radicals.

    1. \(\displaystyle x^{4/5}\)
    2. \(\displaystyle b^{-5/6}\)
    3. \(\displaystyle (pq)^{-2/3}\)
    4. \(\displaystyle 3x{0.4}\)
    5. \(\displaystyle 4z^{-4/3}\)
    6. \(\displaystyle -2x^{0.25}y^{0.75}\)

  2. Write with fractional exponents.

    1. \(\displaystyle \sqrt[3]{x^2}\)
    2. \(\displaystyle 2\sqrt[5]{ab^3}\)
    3. \(\displaystyle \dfrac{-4m}{\sqrt[6]{p^7}}\)
    4. \(\displaystyle \sqrt[3]{(ab)^2}\)
    5. \(\displaystyle \dfrac{8}{\sqrt[4]{x^3}}\)
    6. \(\displaystyle \dfrac{R}{3\sqrt{TK^5}}\)
