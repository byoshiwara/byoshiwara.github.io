A solution to a system of three linear equations in three variables is an ordered triple that satisfies each equation in the system. That triple represents a point where all three planes intersect. If the planes intersect in a single point, the system has a unique solution. If there is no point that lies on all three planes (for instance, if at least two of the planes are parallel), the system is inconsistent. If all three planes are the same, or if they intersect in a line, the system is dependent.
