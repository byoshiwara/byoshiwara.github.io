When we cancel common factors, we are dividing. Because division is the inverse or opposite operation for multiplication, we can cancel common factors, but we cannot cancel common terms.

Finally, the new fractions are like fractions, and we can add them by combining their numerators.

\begin{equation*} \dfrac{2}{3}+\dfrac{3}{4} = \dfrac{8}{12}+\dfrac{9}{12} = \dfrac{17}{12} \end{equation*}

We add or subtract algebraic fractions with the same three steps.
