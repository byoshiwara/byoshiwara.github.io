The ZFP is true even if the numbers \(a\) and \(b\) are represented by algebraic expressions, such as \(x-6\) or \(x+2\text{.}\) For example, if

\begin{equation*} (x-6)(x+2)=0 \end{equation*}

then it must be true that either \(~x-6=0~\) or \(~x+2=0.~\) This is how we can use the ZFP to solve quadratic equations.
