In Example 4, our solution is not the only sinusoidal function that fits the given graph. For example, if we regard the graph as a sine function shifted \(\dfrac{\pi}{3}\) units to the left, we would use the formula \(y = 4\sin (x+\dfrac{\pi}{3})\text{.}\) You can check that the two functions have identical graphs. Usually it is a good idea to choose a formula with a small horizontal shift.
