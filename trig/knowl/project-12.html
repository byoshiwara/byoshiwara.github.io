Algebra Refresher 4.5.

Sketch the graph. Give the equations of any vertical or horizontal asymptotes.

  1. \(\displaystyle f(x) = \dfrac{1}{x - 2}\)

  2. \(\displaystyle g(x) = \dfrac{1}{x + 3} \)

  3. \(\displaystyle h(x) = \dfrac{1}{x^2 - 1}\)

  4. \(\displaystyle H(x) = \dfrac{-1}{(x + 1)^2}\)

  5. \(\displaystyle G(x) = \dfrac{2x + 1}{x}\)

  6. \(\displaystyle F(x) = \dfrac{x}{x - 1}\)

\(\underline{\qquad\qquad\qquad\qquad}\)

Algebra Refresher Answers

  1. \(\displaystyle x = 2,~~y = 0\)

  2. \(\displaystyle x = -3,~~y = 0\)

  3. \(\displaystyle x = -1,~~x = 1~~y = 0\)

  4. \(\displaystyle x = -1,~~y = 0\)

  5. \(\displaystyle x = 0,~~y = 2\)

  6. \(\displaystyle x = 1,~~y = 1\)
