We know that \(\sin^{-1}\left(\dfrac{-\sqrt{3}}{2}\right)\) is an angle whose sine is \(\dfrac{-\sqrt{3}}{2}\text{,}\) and that \(\dfrac{-\pi}{2} \le \theta \le \dfrac{\pi}{2}\text{.}\) Because \(\sin (\theta)\) is negative, we must find an angle in the fourth quadrant, and because \(\sin \left(\dfrac{\pi}{3}\right)=\dfrac{\sqrt{3}}{2}\text{,}\) the reference angle for \(\theta\) is \(\dfrac{\pi}{3}\text{.}\) Thus, the angle we want is \(\theta=\dfrac{-\pi}{3}\text{,}\) and\(\sin^{-1}\left(\dfrac{-\sqrt{3}}{2}\right)=\dfrac{-\pi}{3}\text{.}\)
  2. First note that \(\sin^{-1}(\sin (\pi))\not= \pi\) ! We start by simplifying the expression inside parentheses: \(\alert{\sin (\pi) = 0}\text{,}\) so \(\sin^{-1}(\alert{\sin (\pi)})=\sin^{-1}(\alert{0})\text{.}\) We want an angle whose sine is 0, and which lies in the interval \(\dfrac{-\pi}{2} \le \theta \le \dfrac{\pi}{2}\text{.}\) This angle is 0, so \(\sin^{-1}(\sin (\pi))=0\text{.}\) (It is also true that \(\sin (\pi) = 0\text{,}\) but \(\pi\) is not in the interval \(\dfrac{-\pi}{2} \le \theta \le \dfrac{\pi}{2}\text{.}\))
