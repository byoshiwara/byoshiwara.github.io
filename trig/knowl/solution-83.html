Solution5.19.1
  1. Substitute \(x = 9\) into the expression to find \begin{align*} 2x + \sqrt[3]{x - 1} \amp = 2(\alert{9}) + \sqrt[3]{\alert{9} - 1}\\ \amp = 18 + \sqrt[3]{8} = 20\\ \end{align*}
  2. We must find a value for \(x\) that makes \(2x + \sqrt[3]{x - 1}\) equal to 20. In part (a) we saw that this expression equals 20 when \(x = 9\text{,}\) so the solution of the equation \(2x + \sqrt[3]{x - 1} = 20\) is \(x = 9\text{.}\)
