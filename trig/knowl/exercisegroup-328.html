Exercise Group 71–74.

For Problems 71–74,

  1. Graph the function on the interval \(-2\pi \le\ x \le 2\pi\text{,}\) and use the graph to write the function in a simpler form.

  2. Verify your conjecture algebraically.

71.

\(f(x)=\tan x(\cos x - \cot x)\)

72.

\(g(x)=\csc x - \cot x\cos x\)

73.

\(G(x) = \sin x(\sec x - \csc x)\)

74.

\(F(x) = \dfrac{1}{2}\left(\dfrac{\cos x}{1+\sin x} + \dfrac{1+ \sin x}{\cos x}\right)\)
