Item 3.3.c.
\(r^2 = 24^2 + 10^2 = 676\text{,}\) so \(r = \sqrt{676} = 26.\) Then
\begin{gather*} \cos (\theta) = \dfrac{x}{r} = \dfrac{24}{26} =\dfrac{12}{13}\\ \sin(\theta) = \dfrac{y}{r} = \dfrac{10}{26} =\dfrac{5}{13}\\ \tan(\theta) = \dfrac{y}{x} = \dfrac{10}{24} =\dfrac{5}{12} \end{gather*}
