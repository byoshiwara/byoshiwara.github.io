We compute
\begin{equation*} r=\sqrt{a^2+b^2} = \sqrt{3+1} = 2~~~~\text{and}~~~~\tan (\theta) = \dfrac{1}{\sqrt{3}} \end{equation*}
The angle \(\theta\) is in the first quadrant, so
\begin{equation*} \theta = \tan^{-1} \left(\dfrac{1}{\sqrt{3}}\right) = \dfrac{\pi}{6} \end{equation*}
Thus,\(~~z= 2\left(\cos \left(\dfrac{\pi}{6}\right) + i\sin \left(\dfrac{\pi}{6}\right)\right)\)
