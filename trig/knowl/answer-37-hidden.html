\(\angle 1 = 130\degree\) because vertical angles are equal. \(\angle 2 = 50\degree\) because it makes a straight angle with a \(130\degree\) angle. \(\angle 3 = 65\degree\) because it is a base angle of an isosceles triangle whose vertex angle is \(50\degree\text{.}\) \(\angle 41 = 65\degree\) for the same reason. \(\angle 5 = 25\degree\) because it is complementary to \(\angle 4\text{.}\)
