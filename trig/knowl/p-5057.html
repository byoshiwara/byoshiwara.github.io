When we plot the points in this table, we see that the original graph is traced out again. For example, the point \((\dfrac{7\pi}{6}, -1)\) is the same as the point \((\dfrac{\pi}{6}, 1)\text{,}\) the point \((\dfrac{5\pi}{4}, -\sqrt{2})\) is the same as the point \((\dfrac{\pi}{4}, \sqrt{2})\text{,}\) and so on, around the circle. Thus, the graph of \(r=2\sin \theta\) is a circle, traced twice for \(0 \le \theta \le 2\pi\text{.}\)
