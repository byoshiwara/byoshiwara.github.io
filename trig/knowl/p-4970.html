Each point has infinitely many polar coordinates, because we can add multiples of \(2\pi\) to the value of \(\theta\text{.}\) For example, the point with polar coordinates \((1, \dfrac{\pi}{2})\) also has polar coordinates \((1, \dfrac{5\pi}{2})\) and \((1, \dfrac{-3\pi}{2})\text{.}\) If we allow negative values of \(r\text{,}\) there are even more ways to write the coordinates of a point.
