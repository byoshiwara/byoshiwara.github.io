  1. \(\displaystyle y = \dfrac{5}{12}x\)
  2. \((24, 10)\) satisfies \(y = \dfrac{5}{12}x\text{,}\) that is, the equation \(10 = \dfrac{5}{12}(24)\) is true.
  3. \(r^2 = 24^2 + 10^2 = 676\text{,}\) so \(r = \sqrt{676} = 26.\) Then \(\cos (\theta) = \dfrac{x}{r} = \dfrac{24}{26} =\dfrac{12}{13},~~\sin(\theta) = \dfrac{y}{r} = \dfrac{10}{26} =\dfrac{5}{13}\text{,}\) and \(\tan(\theta) = \dfrac{y}{x} = \dfrac{10}{24} =\dfrac{5}{12}\text{.}\)
