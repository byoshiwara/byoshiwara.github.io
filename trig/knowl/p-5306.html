In the examples above, perhaps you noticed that \(\sin\left(\sin^{-1}(x)\right) = x\) for any value of \(x\) in the domain of the inverse sine. In other words, if we find the inverse sine of a number between \(-1\) and \(1\text{,}\) and then take the sine of the resulting angle, we get the original number back again. The same is true for the cosine and tangent, so that \(\cos\left(\cos^{-1}(x)\right) = x\) and \(\tan\left(\tan^{-1}(x)\right) = x\text{.}\)
