The converse of the Pythagorean theorem is also true. Use the converse to decide whether each of the following triangles is a right triangle. Support your conclusions with calculations.
  1. \(a = 9\text{,}\) \(~ b = 16\text{,}\) \(~ c = 25\)
  2. \(a = 12\text{,}\) \(~ b = 16\text{,}\) \(~ c = 20\)
  3. \(a = \sqrt{8}\text{,}\) \(~ b = \sqrt{5}\text{,}\) \(~ c = \sqrt{13}\)
  4. \(a = \dfrac{\sqrt{3}}{2}\text{,}\) \(~ b = \dfrac{1}{2}\text{,}\) \(~ c = 1\)
