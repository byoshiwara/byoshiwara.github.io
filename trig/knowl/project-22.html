Algebra Refresher 8.3.

  1. Compute \(f(2),~ f(3)\text{,}\) and \(f(2+3)\text{.}\)
  2. For which of the following functions is it true that \(f(a+b) = f(a) + f(b)\) whenever the function values are defined?
  1. \(\displaystyle f(x) = 3x+2\)
  2. \(\displaystyle f(x) = x^2\)
  3. \(\displaystyle f(x) = \sqrt{x}\)
  4. \(\displaystyle f(x) = \dfrac{1}{x}\)
  5. \(\displaystyle f(x) = \abs{x}\)
  6. \(\displaystyle f(x) = 2^x\)
\(\underline{\qquad\qquad\qquad\qquad}\)

    2. Not true
