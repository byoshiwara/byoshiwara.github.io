  1. We can use circular functions of real numbers to describe periodic phenomena.
  2. The domain of a function is the set of all possible input values. The range of a function is the set of all output values for the function.
  3. We can use a graph to solve trigonometric equations, or the inverse trig keys on a calculator or computer. We can find exact values for the solutions of equations involving the special values without using a calculator.
