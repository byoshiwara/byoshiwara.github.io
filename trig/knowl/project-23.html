Algebra Refresher 8.4.

  1. Find a formula for the inverse function.
  2. State the domain and range of the inverse function.
  3. Graph the function and its inverse on the same grid.
  1. \(\displaystyle f(x) = \dfrac{1}{2}x-4\)
  2. \(\displaystyle g(x) = 3x+6\)
  3. \(\displaystyle F(x) = 2+\dfrac{1}{x}\)
  4. \(\displaystyle G(x) = \dfrac{1}{x+5}\)
  5. \(\displaystyle h(x) = \sqrt{x+2}\)
  6. \(\displaystyle H(x) = 3+\sqrt[3]{x}\)
\(\underline{\qquad\qquad\qquad\qquad}\)
Algebra Refresher Answers
    1. \(\displaystyle f^{-1}(x)=2x+8\)
    2. Dom: \((-\infty, \infty)\) Rge: \((-\infty, \infty)\)
    1. \(\displaystyle g^{-1}(x)=\dfrac{1}{3}x-2\)
    2. Dom: \((-\infty, \infty)\) Rge: \((-\infty, \infty)\)
    1. \(\displaystyle F^{-1}(x)=\dfrac{1}{x-2}\)
    2. Dom: \(x \not=0\) Rge: \(y \not=2\)
    1. \(\displaystyle G^{-1}(x)=\dfrac{1}{x}-5\)
    2. Dom: \(x \not=-5\) Rge: \(y \not=0\)
    1. \(\displaystyle h^{-1}(x)=x^2-2\)
    2. Dom: \(x \ge 0\) Rge: \(y \ge -2\)
    1. \(\displaystyle H^{-1}(x)=(x-3)^3\)
    2. Dom: \((-\infty, \infty)\) Rge: \((-\infty, \infty)\)
