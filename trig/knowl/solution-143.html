Solution 8.13.1.

We start by using the Pythagorean Theorem to find the hypotenuse of the triangle.

\begin{equation*} c^2 = 2^2 + 3^2 = 13 \end{equation*}

so \(c = \sqrt{13}\text{.}\) Thus, \(\cos \theta = \dfrac{3}{\sqrt{13}}\) and \(\sin \theta = \dfrac{2}{\sqrt{13}}\text{.}\) Now we can use these values in the double angle identity to find \(\sin 2\theta\text{.}\)

\begin{equation*} \begin{aligned}[t] \sin 2\theta \amp = 2\sin \theta \cos \theta\\ \amp = 2(\dfrac{2}{\sqrt{13}})(\dfrac{3}{\sqrt{13}}) = \dfrac{12}{13}\\ \end{aligned} \end{equation*}
