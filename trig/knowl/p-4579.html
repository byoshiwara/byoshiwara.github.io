\begin{equation*} \begin{aligned}[t] 0.4\tan \theta \amp = -0.5\\ \tan \theta \amp = -1.25\\ \end{aligned} \end{equation*}

Now we can solve for \(\theta\text{.}\)

\begin{equation*} \theta = \tan^{-1}(-1.25)=-0.8960 \end{equation*}

Replacing \(\theta\) by \(3x-0.5\text{,}\) we find the first solution.

\begin{equation*} \begin{aligned}[t] 3x-0.5 \amp = -0.8960\\ 3x\amp = -0.3960\\ x \amp = -0.1320\\ \end{aligned} \end{equation*}
