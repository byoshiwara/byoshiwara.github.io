Algebra Refresher 5.2.

Each of the following "solutions" contains an error. Find the error, and supply a correct solution.
  1. \(\displaystyle \begin{aligned}[t] 3x^2 - 5x\amp = 0 \\ 3x^2 \amp = 5x\\ 3x \amp = 5\\ x \amp = \dfrac{5}{3}\\ \end{aligned}\)
  2. \(\displaystyle \begin{aligned}[t] 4x^2 - 5x\amp =12 \\ x^2 \amp = 3\\ x \amp = \sqrt{3}\\ \end{aligned}\)
  3. \(\displaystyle \begin{aligned}[t] (2x+1)^2\amp = 4 \\ 2x+1 \amp = 2\\ 2x \amp = 1\\ x \amp = \dfrac{1}{2}\\ \end{aligned}\)
  4. \(\displaystyle \begin{aligned}[t] 2x^2 - 4x\amp = 3 \\ 2x(x-2) \amp = 3\\ 2x = 3 \qquad \amp x - 2 = 3\\ x = \dfrac{3}{2} \qquad\amp x = 5\\ \end{aligned}\)
\(\underline{\qquad\qquad\qquad\qquad}\)
Algebra Refresher Answers
  1. We can't divide by \(x.~~x = 0, \dfrac{5}{3}\)
  2. 3 has two square roots. \(~x = \pm\sqrt{3}\)
  3. 4 has two square roots. \(~x = \dfrac{-3}{2}, \dfrac{1}{2} \)
  4. One side must be zero to use the zero-factor principle. No real solutions.
