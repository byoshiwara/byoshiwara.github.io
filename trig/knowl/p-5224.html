We have been using the calculator keys \(\boxed{SIN^{-1}}\text{,}\) \(\boxed{COS^{-1}}\text{,}\) and \(\boxed{TAN^{-1}}\) to find approximate values of \(\theta\) when we know either \(\sin \theta,~ \cos \theta\text{,}\) or \(\tan \theta\text{.}\) For example, if we know that \(\cos \theta = 0.3\text{,}\) then
\begin{equation*} \theta = \cos^{-1}(0.3) \approx 1.2661~ \text{radians} \end{equation*}
