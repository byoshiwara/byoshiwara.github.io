Algebra Refresher 7.2.

  1. Write a formula for \(g(x)\text{.}\)
  2. Graph \(f(x)\) and \(g(x)\) on the same axes.
  1. \(\displaystyle f(x)=x^2,~g(x)=f(x-2)\)
  2. \(\displaystyle f(x)=\sqrt{x},~g(x)=f(x+4)\)
  3. \(\displaystyle f(x)=\dfrac{1}{x},~g(x)=f(x+1)\)
  4. \(\displaystyle f(x)=\dfrac{1}{x^2},~g(x)=f(x-1)\)
  5. \(\displaystyle f(x)=\abs{x},~g(x)=f(x-3)\)
  6. \(\displaystyle f(x)=2^x,~g(x)=f(x+3)\)
\(\underline{\qquad\qquad\qquad\qquad}\)
Algebra Refresher Answers
  1. \(\displaystyle g(x)=(x-2)^2\)
  2. \(\displaystyle g(x)=\sqrt{x+4}\)
  3. \(\displaystyle g(x)=\dfrac{1}{x+1}\)
  4. \(\displaystyle g(x)=\dfrac{1}{(x-1)^2}\)
  5. \(\displaystyle g(x)=\abs{x-3}\)
  6. \(\displaystyle g(x)=2^{x+3}\)
