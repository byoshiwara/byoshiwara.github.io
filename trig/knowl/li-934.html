The angle \(C = 180\degree - (37\degree + 54\degree) = 89\degree\text{.}\) Now we use the Law of Sines to find side \(c\text{.}\) Note that it is safer to use the given side, \(a\text{,}\) rather than the value we calculated for \(b\text{.}\)

\begin{align*} \dfrac{a}{\sin A} \amp = \dfrac{c}{\sin C} \amp\amp \blert{\text{Substitute the known values.}}\\ \dfrac{11}{\sin 37\degree} \amp = \dfrac{c}{\sin 89\degree} \amp\amp \blert{\text{Multiply both sides by} \sin 89\degree.}\\ \dfrac{11}{\sin 37\degree} \cdot \sin 89\degree\amp = c \\ \end{align*}

so \(c \approx 18.28.\)
