Algebra Refresher 5.3.

Write each expression as a single fraction in simplest form.
  1. \(\displaystyle \dfrac{1}{x} + \dfrac{1}{y}\)
  2. \(\displaystyle 1 - \dfrac{1}{x}\)
  3. \(\displaystyle \dfrac{1}{x - 1} + \dfrac{1}{x + 1}\)
  4. \(\displaystyle x - \dfrac{x}{x + 1}\)
  5. \(\displaystyle \dfrac{\dfrac{y}{x} - \dfrac{x}{y}}{\dfrac{y}{x} + 1}\)
  6. \(\displaystyle \dfrac{1 - \dfrac{x^2}{y^2}}{1 + \dfrac{x^2}{y^2}}\)
  7. \(\displaystyle \dfrac{\dfrac{2a}{x}}{1 - \dfrac{a^2}{x^2}}\)
  8. \(\displaystyle \dfrac{\dfrac{a}{x} + \dfrac{b}{y}}{1 - \dfrac{ab}{xy}}\)
\(\underline{\qquad\qquad\qquad\qquad}\)
Algebra Refresher Answers
  1. \(\displaystyle \dfrac{x + y}{xy}\)
  2. \(\displaystyle \dfrac{x - 1}{x}\)
  3. \(\displaystyle \dfrac{2x}{x^2 - 1}\)
  4. \(\displaystyle \dfrac{x^2}{x + 1}\)
  5. \(\displaystyle \dfrac{y - x}{y}\)
  6. \(\displaystyle \dfrac{y^2 - x^2}{y^2 + x^2}\)
  7. \(\displaystyle \dfrac{2ax}{x^2 - a^2}\)
  8. \(\displaystyle \dfrac{ay + bx}{xy - ab}\)
