Exercise Group 9–14.

For Problems 9–14,

  1. Evaluate the expression at the given values of the variable.

  2. Give one solution of the equation.

9.

  1. \(\displaystyle x^3 - 3x^2 + 4;~~~x = -1, 0, 1, 2, 3\)

  2. \(\displaystyle x^3 - 3x^2 + 4 = 0\)

10.

  1. \(\displaystyle \sqrt{x} + \sqrt{2x+1};~~~x = 0, 2, 4, 6\)

  2. \(\displaystyle \sqrt{x} + \sqrt{2x+1} = 5\)

11.

  1. \(\displaystyle \sin \theta + \cos \theta;~~~\theta = 0\degree, 30\degree, 45\degree, 60\degree\)

  2. \(\displaystyle \sin \theta + \cos \theta = \sqrt{2}\)

12.

  1. \(\displaystyle \sin^2 \alpha - \cos \alpha;~~~\alpha = 45\degree, 90\degree, 135\degree, 180\degree\)

  2. \(\displaystyle \sin^2 \alpha - \cos \alpha = 1\)

13.

  1. \(\displaystyle \sin \beta + 2\cos^2 \beta;~~~\beta = 210\degree, 225\degree, 240\degree, 270\degree\)

  2. \(\displaystyle \sin \beta + 2\cos^2 \beta = -1\)

14.

  1. \(\displaystyle 3\cos^2 \phi - \sin^2 \phi;~~~\phi = 270\degree, 300\degree, 315\degree, 330\degree\)

  2. \(\displaystyle 3\cos^2 \phi - \sin^2 \phi = 2\)
