The basic function is \(~f(x)=\dfrac{1}{x^2}~\text{.}\) We subtract 1 from the input, then multiply the output by \(-3\text{,}\) so the transformations are

  1. \(~g(x)=f(x-1)~\text{,}\) so the basic graph is shifted 1 unit to the right
  2. and \(~G(x)=-3g(x)~\text{,}\) so \(g(x)\) is reflected about the \(x\)-axis and stretched vertically by a factor of 3.
