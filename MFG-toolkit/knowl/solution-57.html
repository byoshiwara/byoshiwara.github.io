Solution 2.49.1.
  1. Because \(-2\) lies in the first region, \(f(-2)=4\)
  2. \(1\) lies in the second region, so \(f(\alert{1})=5+2(\alert{1})-(\alert{1})^2 = 6\)
  3. \(2\) lies in the second region, so \(f(\alert{2}) = 5+2(\alert{2})-(\alert{2})^2 = 5\)
