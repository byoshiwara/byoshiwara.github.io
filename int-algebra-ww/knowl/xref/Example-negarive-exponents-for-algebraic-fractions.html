Example 8.4.5.

Write each expression as a single algebraic fraction. \(~\alert{\text{[TK]}}\)
  1. \(\displaystyle x^{-1}-y^{-1}\)
  2. \(\displaystyle (x^{-2}+y^{-2})^{-1}\)

Solution.

  1. We write each power as a fraction, then simplify.
    \begin{align*} x^{-1}-y^{-1} \amp = \dfrac{1}{x} - \dfrac{1}{y} \amp \amp \blert{\text{The LCD is}~xy.}\\ \amp = \dfrac{1}{x} \cdot \blert{\dfrac{y}{y}}- \dfrac{1}{y} \cdot \blert{\dfrac{x}{x}} \amp \amp \blert{\text{Build to the LCD.}}\\ \amp = \dfrac{y-x}{xy} \end{align*}
  2. We simplify the expression inside parentheses first.
    \begin{align*} \left(x^{-2}+y^{-2}\right)^{-1} \amp = \left(\dfrac{1}{x^2} + \dfrac{1}{y^2}\right)^{-1} \amp \amp \blert{\text{Add fractions.}}\\ \amp = \left(\dfrac{y^2+x^2}{x^2y^2}\right)^{-1} = \dfrac{x^2y^2}{x^2+y^2} \end{align*}
    For the last step, remember that a negative exponent indicates the reciprocal of its base.
