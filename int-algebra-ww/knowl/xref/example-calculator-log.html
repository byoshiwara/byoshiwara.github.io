Example 7.3.14.

Approximate the following logarithms to 2 decimal places.
  1. \(\displaystyle \log{(6.5)}\)
  2. \(\displaystyle \log{(256)}\)
Solution.
  1. The keying sequence LOG \(6.5\) )ENTER yields \(.812913566\text{,}\) so \(\log {(6.5)} \approx 0.81\text{.}\)
  2. The keying sequence LOG \(256\) ) ENTER yields \(2.408239965\text{,}\) so \(\log {(256)} \approx 2.41\text{.}\)
